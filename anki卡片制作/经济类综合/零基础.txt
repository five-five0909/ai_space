<div class="front-text">【公式】基本积分表 (幂、指、对、基础三角)</div><br>写出以下函数的不定积分：<br>$x^\alpha (\alpha \neq -1), \frac{1}{x}, a^x, e^x, \sin x, \cos x, \sec^2 x, \csc^2 x$	<div class="latex-box box-green">\[ \int x^\alpha dx = \frac{x^{\alpha+1}}{\alpha+1} + C \]<br>\[ \int \frac{1}{x} dx = \ln|x| + C \quad (\text{绝对值!}) \]<br>\[ \int e^x dx = e^x + C, \quad \int a^x dx = \frac{a^x}{\ln a} + C \]<br>\[ \int \sin x dx = -\cos x + C \quad (\text{负号!}) \]<br>\[ \int \cos x dx = \sin x + C \]<br>\[ \int \sec^2 x dx = \tan x + C \]</div>	396数学::零基础::05_不定积分
