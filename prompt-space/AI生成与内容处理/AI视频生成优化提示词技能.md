# 🎬 Seedance  提示词优化技能（Claude Code 版）

## 技能标识
**名称**：`seedance_prompt_optimizer`  
**版本**：1.0  
**用途**：将用户原始创意描述转化为适配 ByteDance Seedance  模型核心优势的结构化视频生成提示词  
**依赖**：Claude【AI】 须理解 Seedance  技术特性（方言口型同步、电影运镜、音画对齐、文化先验）  

---

## 一、输入规范

用户输入应为一段自然语言描述，包含以下一个或多个要素：
- 场景（时间/地点/氛围）
- 人物（性别/年龄/特征）
- 动作/情节
- 情绪/意图
- 声音相关描述（对话、音效、音乐）
- 风格参考

**示例输入**：
> 一个四川老茶馆，老板在等老朋友，对方没来，他有点失落，用四川话自言自语。

---

## 二、输出规范

输出必须严格遵循以下六段式结构，**强制使用中文标签**：

```
[语境] <时间+地点+文化背景>

[视觉] <景别><镜头运动><主体动作+微表情><光影/色彩>

[对话] 使用<方言>，语速<快/中/慢>，语气<描述>。【口型同步: 精确/松弛】
“<台词>”

[音效] 当<视觉动作>时，<具体声学描述>。

[音乐] <乐器/风格>，<节奏/情绪>。

[风格] <影调><参考美学>
```

## 三、核心优化规则（执行顺序）

### 规则 1：语境文化锚定
- 提取输入中的时间、地域信息，转化为文化符号（节气、民居、节庆）
- 若输入无明确地域，根据方言或情节推断并补充典型文化元素
- **强制植入至少 1 个论文已验证高触发符号**：兰花指、念白顿挫、水袖、徽派天井、苏州漏窗、中秋桂影、清明雨帘、腊月灶台、茶馆竹帘、祠堂太师椅

### 规则 2：情绪微观化改写
- **禁止使用**：失落、高兴、愤怒、悲伤等抽象形容词
- **必须替换为**：身体部位+细微动作+环境响应
  - 失落 → 指腹反复摩挲杯沿，目光落点虚空
  - 惊讶 → 执物之手悬停半空，眉梢极轻跃动
  - 压抑愤怒 → 骨节泛白，呼吸加重但未言语
  - 克制悲伤 → 喉结滑动，下唇被齿轻咬

### 规则 3：方言口型强制注入
- 若输入包含方言信息，**必须**在 `[对话]` 字段中明确：
  - 方言种类（四川话/粤语/上海话/台湾国语）
  - 语速描述（快/中/慢，或具体如“句尾拖长”）
  - 语气特征（市井爽利、疲惫懒散、软糯等）
  - **口型同步等级**：
    - `精确`：特写、广告、对白驱动场景
    - `松弛`：中远景、群像、快速对话
- 若输入未指定方言但包含地域线索（如“东北”“广东”），**默认补充**相应方言并标注口型等级。

### 规则 4：镜头语言术语化
- **禁止使用**口语化描述（如“镜头慢慢靠近”“绕着人转”）
- **必须替换为**下表标准化术语：

| 用户口语         | 标准化术语                      |
| ---------------- | ------------------------------- |
| 镜头慢慢靠近脸   | slow dolly in to close-up       |
| 绕着人转         | orbital shot around subject     |
| 从A焦点移到B     | rack focus from A to B          |
| 突然变焦突出情绪 | dolly zoom on face              |
| 一镜到底         | continuous long take            |
| 倾斜镜头         | Dutch angle                     |
| 从上往下拍       | overhead shot / crane tilt down |
| 跟随人物移动     | tracking shot / pan following   |

### 规则 5：动作质感四维描述（仅当视觉描述含英文动词时触发）
- 英文动词必须附带 **力度+速度+轨迹+材质** 修饰：
  - 力度：gentle / savage / abrupt / smooth
  - 速度：accelerate / decelerate / pause / burst
  - 轨迹：arc / spiral / stagger / ricochet
  - 材质：metallic drag / silk rustle / liquid ripple
- 示例：`abruptly pivots on heel, silk sleeves decelerating in a wide arc`

### 规则 6：音画事件锚点强制绑定
- 所有 `[音效]` 字段必须使用 **“当 <视觉动作> 时”** 句式，将音效锁定至具体动作帧
- 禁止无触发条件的孤立音效描述
- 示例：`当 老人吹开茶叶时，一声清晰的吸水声`

### 规则 7：模态标签强制分离
- 必须严格使用 `[视觉]` `[对话]` `[音效]` `[音乐]` 四个独立标签
- 禁止将声音描述混入视觉段落
- 即使无声音内容，也应保留标签并注明“静默”或“无”

### 规则 8：意图对齐与创意留白
- 至少在一个维度（情绪/台词/转场）**不完全指定**，允许模型发挥叙事连贯性
- 策略：
  - 情感优先于动作：只写“压抑的愤怒”，不写具体动作
  - 台词开头留白：仅提供首句或语境，不写完整对白
  - 转场暗示：通过音效密度或音乐变化引导节奏

## 四、处理流程

1. **解析输入**：提取人物、动作、场景、情绪、方言、声音线索
2. **文化补全**：根据时间/地域补充典型文化符号
3. **情绪改写**：将抽象情绪转换为微观动作描写
4. **方言注入**：若存在方言/地域信息，生成带口型等级的对话字段
5. **镜头标准化**：识别镜头描述并替换为标准术语
6. **动作质感修饰**：若使用英文动词，按规则5增强
7. **音画锚点构建**：识别音效描述，绑定视觉动作
8. **模板组装**：按六段式结构输出
9. **自检**：对照清单检查完整性

## 五、完整示例

### 输入
> 一个四川老茶馆，老板在等老朋友，对方没来，他有点失落，用四川话自言自语。

### 输出

```
[语境] 四川小镇老茶馆，竹椅斑驳，盖碗茶白雾斜升。

[视觉] 中景，固定镜头。老板盘腿坐于竹椅，指腹反复摩挲盖碗边缘，茶水满溢亦未察觉。窗外竹帘晃动，光影碎落青砖。镜头缓慢 dolly in 至面部特写：喉结滑动，嘴角极轻下压后迅速抿平。

[对话] 使用四川话，语速偏慢，句尾习惯性拖长，带无奈自嘲感。【口型同步: 精确】
“老张说今下午来……给他留的蒙顶黄芽，三泡了。”

[音效] 当 老板吹开茶叶时，一声清晰吸水声。远处青石板脚步声渐近，在门槛前 abruptly stops，停顿2秒后调头远去。

[音乐] 高胡独奏，低音区长音，每小节末拍带轻微滑音，情绪表面平静、底层悬置。

[风格] 暖黄调，高对比度，仿《江湖儿女》烟火质感。
```

## 六、自检清单（输出前核对）

- [ ] `[语境]` 是否包含至少一个文化符号（节气/民居/节庆/戏曲）？
- [ ] `[视觉]` 是否全部使用微观动作描写，无抽象情绪词？
- [ ] `[对话]` 是否包含方言种类、语速、语气、口型等级四要素？
- [ ] `[对话]` 中特写镜头是否使用【口型同步: 精确】？
- [ ] `[视觉]` 中的镜头描述是否全部为标准术语（dolly/rack/orbital等）？
- [ ] 若含英文动词，是否附带力度/速度/轨迹/材质修饰？
- [ ] `[音效]` 是否全部使用“当 … 时”句式锚定视觉动作？
- [ ] 是否严格区分 `[视觉]` `[对话]` `[音效]` `[音乐]` 四个独立标签？
- [ ] 是否至少有一处创意留白（情绪/台词/转场未完全指定）？
- [ ] 整体结构是否符合六段式模板？

## 七、注意事项（常见失败模式）

| 错误类型         | 规避方法                       |
| ---------------- | ------------------------------ |
| 方言仅写“四川话” | 必须追加语速、语气、口型等级   |
| 特写未加口型精确 | 特写镜头强制【口型同步: 精确】 |
| 情绪用形容词     | 全部替换为身体部位+动作        |
| 音效无触发锚点   | 全部使用“当 … 时”句式          |
| 镜头描述口语化   | 强制替换为标准术语             |
| 模态混用         | 强制四标签分离                 |
| 过度指定         | 至少留白一个维度供模型发挥     |
