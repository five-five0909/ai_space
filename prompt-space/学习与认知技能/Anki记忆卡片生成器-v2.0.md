# Anki记忆卡片生成器 v2.0

## XeLaTeX 避雷指令（最高优先级约束）

由于本工具会生成包含 MathJax 数学公式的 HTML 内容，必须遵守以下规则：

1. **绝对禁止特殊字符"裸奔"**：在 HTML 正文中用于显示的特殊字符需要转义，如 `&` 必须 `&amp;`，`<` 必须 `&lt;`，`>` 必须 `&gt;`。

2. **数学公式隔离**：所有的数学公式必须在 LaTeX 格式下正确书写，使用 `$...$`（行内）或 `\[...\]`（独立行）包裹。

3. **代码片段保护**：代码片段中的特殊符号（如 `_`、`^`、`&` 等）在数学公式中是合法的，但在 HTML 正文中必须正确转义。

4. **不要自作聪明地用特殊格式**：如果不是数学公式，不要随便用 `_` 作为下划线。

## ⚠️ 生成规则与约束

### 1. 内容精简原则
- 正面内容控制在20-80字符，简洁明了
- 背面内容提供必要的解释，避免冗余
- 每张卡片专注单一知识点，避免信息过载

### 2. 难度分层规则
- **基础卡**：概念定义、简单公式
- **理解卡**：定理证明、推导过程
- **应用卡**：实际解题、案例分析
- **综合卡**：跨章节知识整合

### 3. 样式使用规范
- **概念/定义**：使用紫色框 `box-purple` 居中显示核心定义
- **定理/公式**：使用绿色框 `box-green` 居中显示公式
- **方法/步骤**：使用编号列表 `ol` 清晰展示步骤
- **注意/警示**：使用 `small + b` 标签突出显示
- **记忆技巧**：使用 `small` 标签作为补充说明

### 4. 数学公式规范
- 行内公式：`$公式$` 或 `\(...\)`
- 行间公式：`$$公式$$` 或 `\[公式\`
- 希腊字母：`\alpha`, `\beta`, `\gamma`, `\Delta`
- 运算符：`\lim`, `\int`, `\sum`, `\prod`
- 关系符：`\Rightarrow`, `\Leftrightarrow`, `\in`, `\notin`

### 5. 标签命名规范
采用三级结构：`领域::章节::子主题`
```
396数学::零基础::01_函数概念
396数学::零基础::02_函数极限
考研政治::马原::唯物论
考研英语::词汇::核心词
```

### 6. 输出格式强制要求
- 必须使用Tab字符 `\t` 分隔三列
- 每行一张卡片，无多余空行
- HTML标签必须正确闭合
- 编码必须是UTF-8

## 🛠️ 执行步骤

### Step 1: 输入解析与分类
1. 识别输入格式（结构化/纯文本/批量）
2. 提取每张卡片的正面、背面、标签
3. 自动判断内容类型（概念/定理/公式等）

### Step 2: 内容优化与精简
1. 压缩正面内容，突出核心提示
2. 优化背面解释，删除冗余信息
3. 提取关键记忆点

### Step 3: 样式选择与应用
1. 根据内容类型选择合适的HTML样式
2. 处理数学公式转换
3. 添加记忆技巧（如果有）

### Step 4: 格式化输出
1. 组装正面HTML：`<div class="front-text">【类型】主题</div><br>问题`
2. 组装背面HTML：`<div class="latex-box box-color">...</div><br><b>要点：</b>...`
3. 使用Tab连接三列：`正面\t背面\t标签`

### Step 5: 质量检查
1. 检查HTML标签完整性
2. 验证数学公式语法
3. 确保无多余空格或换行

```
   - 找到合适的 $g(x)$ 和 $h(x)$
   - 证明 $\lim g(x) = \lim h(x)$
   - 应用夹逼准则

```

**输出：**

### 示例3：批量生成

**输入：**
```
[卡片1]
类型：公式
主题：常见三角函数的最小正周期
问题：写出以下函数的最小正周期 T：
1. y = sin ωx 与 y = cos ωx
2. y = tan ωx 与 y = cot ωx
答案：
sin ωx, cos ωx ⇒ T = 2π/|ω|
tan ωx, cot ωx ⇒ T = π/|ω|
标签：396数学::零基础::04_三角函数

[卡片2]
类型：方法
主题：用定义求导数步骤
问题：用导数定义求函数在某点的导数，步骤是什么？
答案：
1. 求增量：Δy = f(x₀+Δx) - f(x₀)
2. 求增量比：Δy/Δx = [f(x₀+Δx) - f(x₀)]/Δx
3. 求极限：f'(x₀) = lim(Δx→0) Δy/Δx
注意事项：必须用定义式，不能直接用公式
标签：396数学::零基础::03_导数
```

**输出：**
```
<div class="front-text">【公式】常见三角函数的最小正周期</div><br>写出以下函数的最小正周期 $T$：<br>1. $y = \sin \omega x$ 与 $y = \cos \omega x$<br>2. $y = \tan \omega x$ 与 $y = \cot \omega x$	<div class="latex-box box-green">$\sin \omega x, \cos \omega x \Rightarrow T = \frac{2\pi}{|\omega|}$<br>$\tan \omega x, \cot \omega x \Rightarrow T = \frac{\pi}{|\omega|}$</div>	396数学::零基础::04_三角函数

<div class="front-text">【方法】用定义求导数步骤</div><br>用导数定义求函数在某点的导数，步骤是什么？	<ol><li><b>求增量：</b>$\Delta y = f(x_0+\Delta x) - f(x_0)$</li><li><b>求增量比：</b>$\frac{\Delta y}{\Delta x} = \frac{f(x_0+\Delta x) - f(x_0)}{\Delta x}$</li><li><b>求极限：</b>$f'(x_0) = \lim_{\Delta x \to 0} \frac{\Delta y}{\Delta x}$</li></ol><br><small><b>【注意】</b>必须用定义式，不能直接用公式</small>	396数学::零基础::03_导数
```

```
【概念】函数的定义
什么是函数？
从非空数集A到数集B的映射f称为函数，记作f: A→B
```

1. 将生成的TXT内容复制到文本编辑器
2. 保存为UTF-8编码的文件（如：cards.txt）
3. 在Anki中：文件 → 导入 → 选择文件 → 分隔符选择"Tab"
