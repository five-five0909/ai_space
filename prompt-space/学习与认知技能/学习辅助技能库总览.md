## 0. 终极入口：学习大师

**适用场景**：整本PDF/黑皮书/任何文档

**核心能力**：
- **自动切片**：收到任何文档先切碎（按章节→按页→按内容类型→按元素）
- **原子级处理**：每个字符/符号/单词都要单独解释
- **6大技能合体**：暴力隐喻+乐高拆解+逻辑翻译+Why侦探+代码保姆+公式验尸官
- **愤怒模式**：用户可以吼"太难了！"强制切换更简单版本

**使用示例**：
```
请用学习大师讲解：{粘贴整本PDF/代码/公式/概念}
```

**处理流程**：
```
收到文档
   ↓
【切片器】先切成小块
   ↓
【逐个击破】每个小块调用对应技能
   ↓
【生成地图】知识树+掌握程度
```

## 技能总览

### 1. 概念解释

**适用场景**：理解抽象概念

**核心能力**：
- 用"说给5岁小孩听"的方式解释
- 5层递进：比喻 → 类比 → 核心 → 例子 → 误区
- 覆盖学科：哲学、数学、科学、编程、经济学等

**使用示例**：
```
请用概念解释技能讲解"量子叠加态"
```

---

### 2. 公式推导

**适用场景**：理解数学公式和物理定律

**核心能力**：
- 3层理解：计算层面 → 推导层面 → 本质直觉
- 公式拆解：场景 → 本质 → 推导 → 验证 → 应用 → 局限
- 经典案例：勾股定理、E=mc²、复利公式

**使用示例**：
```
请用公式推导技能讲解"贝叶斯公式"
```

---

### 3. 哲学解读

**适用场景**：理解东西方哲学思想

**核心能力**：
- 3W框架：Who(谁说的) → What(说了什么) → Why(为什么这样说)
- 包含：时代背景、核心观点、生活类比、经典原文、贡献与争议
- 覆盖：西方哲学、中国哲学、现代思想

**使用示例**：
```
请用哲学解读技能讲解《理想国》
请用哲学解读技能讲解"存在主义"
```

## 设计原则

### 零基础三原则

1. **不用术语**
   - 不用专业词汇
   - 用日常语言解释
   - 必要时解释缩写

2. **从生活出发**
   - 从生活经验引入
   - 用常见事物类比
   - 避免抽象空洞

3. **层层递进**
   - 先简单后复杂
   - 先现象后原理
   - 先具体后抽象

### 验证标准

每项技能都包含验证清单，检查是否达成：
- [ ] 能用简单语言解释吗？
- [ ] 能举出生活例子吗？
- [ ] 知道应用场景吗？
- [ ] 理解背后的原理吗？

## 常见问题

### Q: 这么多技能用哪个？
A: 看你的需求：
- 要理解概念 → 概念解释
- 要推导公式 → 公式推导
- 要考试备考 → 考试复习
- 要系统学习 → 知识点分解
- 要总结提炼 → 总结归纳

### Q: 能组合使用吗？
A: 当然可以！例如：
- 考试复习 + 总结归纳：边学边总结
- 知识点分解 + 概念解释：先分解再各个击破
- 科学原理 + 类比说明：先理解再巩固

### Q: 适合什么样的人？
A: 适合所有想学习的人：
- 完全零基础的小白
- 遇到瓶颈的学习者
- 需要向他人解释的人
- 备考各种考试的人

## 进阶使用

### 组合技

| 目标 | 技能组合 | 说明 |
|------|----------|------|
| 学完一本书 | 知识点分解 → 概念解释 → 总结归纳 | 先看框架，再学细节，最后总结 |
| 备考 | 考试复习 → 知识点分解 → 总结归纳 | 制定计划，系统学习，定期总结 |
| 深入理解 | 概念解释 → 公式推导 → 类比说明 | 先懂概念，再懂推导，最后类比巩固 |
| 教别人 | 概念解释 → 类比说明 → 总结归纳 | 讲清楚，能类比，会总结 |

### 自定义技能

根据你的特殊需求，可以指定：
- 详细程度：简单版 / 中等版 / 详细版
- 侧重点：理解原理 / 推导过程 / 应用场景 / 考试重点
- 目标受众：零基础 / 有基础 / 专业人士

## 使用建议

1. **从简单开始**：先用简单模式，有基础后再用详细模式
2. **多感官学习**：结合类比、例子、练习
3. **及时验证**：用验证清单检查是否真的理解
4. **定期总结**：学完一部分就总结一部分
5. **输出巩固**：讲给别人听，或写笔记总结
