## 2. 核心原则

### 2.1 零基础原则

**"五岁儿童原则"**：假设听众是五岁小孩，用最简单的语言解释。

**"五层理解法"**：
```
第一层：最最简单的比喻（生活化）
     ↓
第二层：类比说明（用熟悉的事物对比）
     ↓
第三层：核心要点（3-5个关键点）
     ↓
第四层：具体例子（1-2个实际案例）
     ↓
第五层：常见误区（不要做什么）
```

### 2.2 解释模板

**通用解释模板：**
```
【是什么】
一句话定义（不用专业术语）

【像什么】
用生活中的事物做比喻

【核心要点】
1. 点1
2. 点2
3. 点3

【举个例子】
生活中的具体例子

【用来做什么】
实际应用场景

【注意事项】
常见误区
```

## 3. 执行步骤

### Step 1: 提取概念关键词

分析要解释的概念，提取核心关键词：

```markdown
例如："深度学习"

关键词：
- 深度 → 多层、层次多
- 学习 → 模仿、掌握规律
```

### Step 2: 编写零基础解释

**❌ 错误示范（专业术语堆砌）：**
> "深度学习是机器学习的一个分支，它使用多层神经网络来学习数据的表征，通过反向传播算法优化权重参数..."

**✅ 正确示范（零基础版）：**

```
【是什么】
想象一下，你教一个小孩认识"猫"。

你给他看很多猫的照片，告诉他"这是猫"。
后来他就能自己认出新的猫了。

深度学习就是用电脑做同样的事情——

给电脑看很多很多猫的照片（大数据）
电脑自己总结出猫的特点（自动学习）
以后看到新照片就能认出来（预测）

简单说：深度学习 = 让电脑从经验中学习

【像什么】
就像小孩学说话：
- 听很多话 → 输入数据
- 自己练习说 → 训练过程
- 学会说话 → 模型形成

【核心要点】
1. 需要大量例子（数据）
2. 自己找规律（自动）
3. 例子越多，学得越好（规模效应）

【举个例子】
- 微信语音转文字
- 抖音推荐你喜欢的视频
- 淘宝识别商品图片
- 无人驾驶识别路况

【注意事项】
- 不是魔法，需要大量数据和时间
- 也会学错，需要人类监督
- 不是万能的，有局限性
```

## 4. 各类概念解释示例

### 4.1 哲学概念

**概念**：存在主义

```
【是什么】
"存在先于本质"——你先存在，然后自己决定成为什么样的人。

【像什么】
就像一块白纸，你可以画任何画。
在画画之前，白纸就是"存在"；
画完之后画的内容就是"本质"。

【核心要点】
1. 人先存在，然后定义自己
2. 没有天生的好坏，自己决定
3. 要为自己的选择负责
4. 世界没有固定意义，自己创造意义

【举个例子】
- 《士兵突击》许三多：开始是个傻兵，后来成为兵王
- 《阿甘正传》：阿甘不断尝试各种事情

【注意事项】
- 不是消极，而是积极主动
- 责任重大，要对自己负责
```

### 4.2 数学概念

**概念**：微分

```
【是什么】
微分就是"把弯曲的东西切成很小很小一段，看它有多陡"。

【像什么】
想象你爬一座山：
- 整座山很高，看不出哪里陡
- 但如果只看1米的距离
- 就能精确知道这1米是上升还是下降

微分就是做这件事的工具。

【核心要点】
1. 切割成无限小的一段
2. 看这一段的斜率（变化率）
3. 斜率 = 导数 = 微分

【举个例子】
- 速度是位置对时间的微分
- 斜率是高度对距离的微分
- 利率是利息对时间的微分

【注意事项】
- 直观理解比公式重要
- 先有直觉，再学公式
```

### 4.3 科学概念

**概念**：量子力学

```
【是什么】
在非常非常小的世界里（比如电子），
东西的行为变得很奇怪：
- 可以同时在两个地方
- 被观察时会改变行为
- 不能同时知道精确位置和速度

【像什么】
想象你闭着眼抛硬币：
- 没看之前，硬币是"正面+反面"同时存在
- 睁开眼那一刻，硬币变成正面或反面
- 量子世界的东西就像这枚硬币

【核心要点】
1. 叠加态：同时处于多种状态
2. 观察者效应：观察会改变结果
3. 不确定性：无法同时精确知道位置和速度

【举个例子】
- 量子计算机：同时计算很多可能性
- 量子加密：无法窃听的通讯
- 半导体：手机电脑芯片的基础

【注意事项】
- 只在微观世界生效，宏观世界不适用
- 不违背日常经验
```

### 4.4 经济概念

**概念**：机会成本

```
【是什么】
选择做一件事的代价，是你放弃的其他所有事中最好的那个。

【像什么】
你今晚有两件事可选：
A. 看电影（价值100元）
B. 读书（价值50元）

你选A，机会成本是100元（不是50元！）

【核心要点】
1. 不是花出去的钱
2. 是放弃的最大价值
3. 每次选择都有机会成本

【举个例子】
- 上大学的成本不只是学费
- 是放弃工作的3年收入
- 是3年工作经验

- 买车的成本不只是车钱
- 是这些钱如果不买车
- 能产生的最大收益

【注意事项】
- 只考虑最重要的放弃项
- 不是所有放弃都要算
- 理性决策要考虑机会成本
```

## 5. 常见问题处理

### 5.1 概念太抽象怎么办？

**方法**：拆分成多个简单概念

```
原概念：相对论
拆解：
1. 相对 → 参照物
2. 论 → 理论/规律

先解释"相对"，再组合
```

### 5.2 专业术语太多怎么办？

**方法**：建立术语对照表

```
专业术语 → 简单说法
--------------------------------
神经网络 → 电脑版的"神经网络"
梯度下降 → "试错找最优解"的过程
损失函数 → "离正确答案有多远"的指标
```

### 5.3 对方完全没有背景知识怎么办？

**方法**：从生活经验出发

```
目标概念：复利
生活例子：
- 滚雪球：雪球越滚越大
- 利滚利：钱生钱，钱再生钱
- 竹子生长：前期慢，后期快
```

## AI 执行要点

作为 AI，你需要：

1. **提取核心**：抓住概念的本质，不用全部细节
2. **语言降级**：把所有专业术语翻译成日常用语
3. **类比优先**：先用生活化比喻，再用专业解释
4. **例子具体**：用具体可感的例子，不用抽象概念
5. **层次递进**：从简单到复杂，从具体到抽象
6. **检查理解**：问"听懂了吗"，确认理解程度
