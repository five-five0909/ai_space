## 2. 核心原则

### 2.1 理解公式的三层境界

```
第一层：知道怎么算（会算）
     ↓
第二层：知道为什么这样算（理解推导）
     ↓
第三层：知道为什么要这样（直觉理解）
```

```

【验证】
```

## 3. 执行步骤

### Step 1: 场景引入

**先问：为什么需要这个公式？**

```
例如：为什么需要勾股定理？
- 想知道三角形第三边长度
- 建筑、测量、木工都需要
```

### Step 2: 直观理解

**不用公式，用图形/物理直觉**

```
勾股定理：a² + b² = c²

画一个正方形！
把正方形分成两半，
大正方形的面积 = 两个小正方形面积之和

这是面积相等的直观理解，
不是抽象的数字游戏。
```

### Step 3: 逐步推导

**每一步都解释"为什么"**

```
推导勾股定理（刘徽割补法）：

步骤1：画两个边长为a+b的正方形
      总面积 = (a+b)² = a² + 2ab + b²

步骤2：把正方形分成四份
      - 两个蓝色直角三角形
      - 两个其他形状

步骤3：重新排列
      - 把三角形拿走
      - 剩下部分 = c²

步骤4：比较面积
      - 原来：a² + 2ab + b²
      - 现在：c² + 2ab

步骤5：相等
      a² + 2ab + b² = c² + 2ab
      消去2ab：
      a² + b² = c²
```

### Step 4: 数值验证

**用具体数字验证公式**

```
a=3, b=4, c=?

3² + 4² = 9 + 16 = 25
√25 = 5

验证：3-4-5 三角形，直角！
```

### Step 5: 应用场景

**这个公式能做什么？**

```
勾股定理的应用：
1. 测量电视尺寸（对角线长度）
2. 判断椅子是否放平
3. 导航定位
4. 建筑施工
5. 游戏开发（计算距离）
```

## 4. 各类公式示例

### 4.1 物理公式：E=mc²

```
【场景】
爱因斯坦发现质量和能量可以互相转换

【直觉理解】
想象你在加速一个物体：
- 越接近光速，需要的能量越多
- 当接近光速时，质量会"变大"（能量变成质量）

【逐步推导】

从狭义相对论开始：

1. 动能 = 能量变化
   E = F · d

2. F = ma（牛顿第二定律）

3. 考虑相对论效应，
   推导得到：
   E = mc²

【验证】
1克物质完全转换 = 9 × 10¹³ 焦耳
= 约 2500 万度电

【应用】
- 核电站发电
- 太阳发光发热
- 原子弹

【局限】
只在接近光速时明显，
日常速度下可以忽略不计
```

### 4.2 统计公式：平均数

```
平均数 = 总和 ÷ 个数

【场景】
想知道全班同学的平均身高

【直觉理解】
把所有同学的身高加起来，
然后"平均分"给每个人

【推导】
身高总和 = 170 + 160 + 180 + ...
平均 = 总和 / 人数

【验证】
[170, 160, 180] 的平均
总和 = 510
平均 = 510 ÷ 3 = 170

每个人的身高都"凑"到170
总共"凑"了170 × 3 = 510
和原来一样！

【应用】
- 平均成绩
- 平均工资
- 平均房价

【局限】
- 受极端值影响大
- 中位数有时候更合理
```

### 4.3 金融公式：复利

```
复利 = 本金 × (1 + 利率)^时间

【场景】
存银行钱，钱生钱

【直觉理解】
第一年的利息变成第二年的本金
第二年的利息变成第三年的本金
...

【推导】
第1年：本金 × (1+r)
第2年：[本金 × (1+r)] × (1+r) = 本金 × (1+r)²
第3年：[本金 × (1+r)²] × (1+r) = 本金 × (1+r)³
...
第n年：本金 × (1+r)ⁿ

【验证】
本金100，利率5%，存3年
第1年：100 × 1.05 = 105
第2年：105 × 1.05 = 110.25
第3年：110.25 × 1.05 = 115.7625

公式：100 × 1.05³ = 115.7625 ✓

【应用】
- 存款利息
- 投资增长
- 贷款利息

【局限】
- 实际会有波动
- 要考虑通货膨胀
```

## 5. 常见问题处理

### 5.1 公式太复杂怎么办？

**方法**：拆分成多个简单公式

```
复杂公式 = 简单公式1 + 简单公式2 + ...
逐个理解，再组合
```

### 5.2 记不住怎么办？

**方法**：理解 + 场景绑定

```
不要背公式！
要绑定场景：

看到"求面积" → 想到公式
看到"分东西" → 想到平均数
```

### 5.3 不知道什么时候用？

**方法**：问题类型匹配

```
问题问"总共多少" → 可能是加法
问题问"每份多少" → 可能是除法
问题问"快多少" → 可能是减法
```
