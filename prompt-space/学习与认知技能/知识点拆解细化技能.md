## 2. 核心原则

### 2.1 分解的基本思路

```
复杂知识 → 简单知识
     ↓
简单知识 → 基础概念
     ↓
基础概念 → 已有认知
     ↓
连接成线 → 形成体系
```

### 2.2 分解模板

```
【整体概览】
这门学科/章节是讲什么的？
要解决什么问题？

【前置知识】
学习这个需要什么基础？
如果没有，怎么补？

【核心模块】
这门学科分成几个部分？
每个部分讲什么？

【知识地图】
模块之间的关系是什么？
学习顺序是什么？

【关键概念】
最重要的3-5个概念是什么？

【应用场景】
学会了这个能做什么？
```

## 3. 执行步骤

### Step 1: 画知识地图

**先看森林，再看树木**

```
第一步：找到知识骨架
- 看目录
- 看大纲
- 找知识结构图

第二步：识别模块
- 分成几大块？
- 每块的核心是什么？

第三步：找到连接
- 模块之间什么关系？
- 学习顺序是什么？
```

### Step 2: 识别前置知识

**补齐短板再出发**

```
学习这个需要什么？

例如：学微积分需要
- 函数
- 极限
- 导数（微分的一部分）

如果不会函数，先补函数！
如果不懂极限，先补极限！
```

### Step 3: 从基础开始

**打牢地基再盖楼**

```
学习顺序：
1. 基础概念（什么是...）
2. 基本规则（怎么操作...）
3. 简单应用（用在哪里...）
4. 复杂应用（综合运用...）
5. 进阶内容（更高阶...）
```

### Step 4: 建立连接

**把散落的知识串起来**

```
连接方式：
1. 因果关系（A导致B）
2. 包含关系（A包含B）
3. 对比关系（A vs B）
4. 先后关系（A之后学B）
5. 应用关系（A用于B）
```

## 4. 分解示例

### 4.1 整门学科分解

**例子：高中物理**

```
【整体概览】
物理 = 研究物体运动和相互作用规律的学科

要解决问题：
- 东西为什么动？
- 东西怎么动？
- 为什么会相互作用？

【前置知识】
- 数学（计算）
- 语文（理解题意）

【核心模块】

模块一：力学（40%）
- 运动描述
- 力与运动
- 功和能
- 动量

模块二：电磁学（30%）
- 电场
- 电路
- 磁场
- 电磁感应

模块三：热学（15%）
- 分子动理论
- 固体液体气体
- 热力学定律

模块四：光学（10%）
- 几何光学
- 物理光学

模块五：近代物理（5%）
- 原子结构
- 原子核

【知识地图】
力学 → 电磁学（力和电的联系）
力学 → 热学（能量转化）
力学 → 振动（简谐运动）

【学习顺序】
1. 力学（最基础）
2. 热学（相对独立）
3. 电磁学（需要力学基础）
4. 光学（独立模块）
5. 近代物理（综合）

【关键概念】
1. 力（F）
2. 能量（E）
3. 动量（p）
4. 场（电场/磁场）
5. 能量守恒

【应用场景】
- 解释生活现象
- 工程技术
- 科学研究
```

### 4.2 单章分解

**例子：高中物理 - 牛顿运动定律**

```
【整体概览】
牛顿运动定律 = 解释物体"为什么这样运动"

要解决问题：
- 物体为什么保持静止？
- 物体为什么加速/减速？
- 作用力与反作用力？

【前置知识】
- 描述运动的物理量（速度、加速度）
- 力的概念

【核心内容】

第一定律（惯性定律）
- 核心：物体不受力，保持原状态
- 理解：有力才改变，没力就保持

第二定律（F=ma）
- 核心：力改变运动状态
- 理解：力大=加速度大；质量大=难改变

第三定律（作用反作用）
- 核心：力成对出现
- 理解：你推墙，墙也推你

【学习顺序】
1. 第一定律（概念）
2. 第二定律（核心公式）
3. 第三定律（对称性）
4. 综合应用（解题）

【关键概念】
1. 惯性
2. 加速度
3. 质量（不是重量！）
4. 作用力与反作用力

【常见误区】
- 质量=重量（错！质量是物体本身，重量是重力）
- 有力一定有速度（错！有力一定有加速度）
- 静止物体不受力（错！静止可能受平衡力）
```

### 4.3 概念分解

**例子：微积分 - 导数**

```
【整体概览】
导数 = 变化率

要解决问题：
- 事物变化有多快？
- 曲线在某点有多陡？

【前置知识】
- 函数（什么是函数）
- 极限（无限接近但不等于）

【概念分解】

概念1：变化率
- 速度 = 位置变化率
- 斜率 = 高度变化率
- 增长率 = 数量变化率

概念2：瞬时变化率
- 平均变化率 ≈ 瞬时变化率（当时间足够短）
- 导数 = 瞬时变化率

概念3：几何意义
- 导数 = 曲线在该点的切线斜率

【公式】
f'(x) = dy/dx = lim(Δx→0) [f(x+Δx)-f(x)]/Δx

【简单解释】
- 不用公式：切线有多陡
- 用比喻：开车时速度表显示的速度

【应用场景】
- 速度、加速度
- 经济中的边际成本
- 物理中的各种变化率
```

## 5. 分解技巧

### 5.1 遇到完全陌生的领域

```
方法：从生活经验出发

例如：学量子力学

生活经验：
- 日常东西都是确定的
- 位置是确定的

量子世界：
- 东西是不确定的
- 只能知道概率

从不同点引入：
- 不是"更快的计算机"
- 而是"另一个世界的规则"
```

### 5.2 知识太多记不住

```
方法：记住骨架

知识 = 树
- 树干 = 核心概念
- 树枝 = 展开内容
- 树叶 = 具体细节

考试考树干和主要树枝
不用记住所有树叶

重点：
- 记住核心概念
- 记住模块关系
- 记住基本公式
```

### 5.3 学了不理解

```
方法：回到更基础

不理解 → 回到更基础的概念

例：不懂电磁感应
↓ 需要理解
不懂电生磁
↓ 需要理解
不懂磁场
↓ 需要理解
不懂磁铁
↓ 回到生活经验
```

### 5.4 知识碎片化

```
方法：建立连接

连接问题：
- 这个和什么有关系？
- 这个能推出什么？
- 什么情况下用这个？

建立知识网：
- A → B（因果）
- A + B = C（组合）
- A 在 B 中的应用（应用）

做思维导图
```

## 6. 学习路径设计

### 6.1 零基础入门路径

```
第1天：概览
- 了解这门学科是什么
- 有什么用
- 整体框架

第2-3天：前置知识
- 补齐基础概念
- 建立基本认知

第4-7天：核心模块1
- 模块1的核心概念
- 基本公式
- 简单应用

第8-10天：核心模块2
...

第N天：综合应用
- 模块之间的关系
- 复杂问题求解
```

### 6.2 进阶路径

```
入门 → 理解 → 熟练 → 精通

入门：
- 知道有什么
- 了解基本概念
- 能做简单题

理解：
- 知道为什么
- 理解原理
- 能做中等题

熟练：
- 知道怎么用
- 快速解题
- 能做难题

精通：
- 融会贯通
- 能讲给别人
- 能创新应用
```
