## 2. 核心原则

### 2.1 高效复习的"三要三不要"

```
三要：
1. 要理解，不要死记
2. 要重复，不要一遍过
3. 要输出，不要只看

三不要：
1. 不要从头到尾读课本
2. 不要抄书式的做笔记
3. 不要盲目刷题
```

### 2.2 复习金字塔

```
    主动输出（90%记住）
       ↑
    教授他人（70%记住）
       ↑
    实践应用（60%记住）
       ↑
    讨论交流（50%记住）
       ↑
    演示操作（30%记住）
       ↑
    看图学习（20%记住）
       ↑
    听讲（5%记住）
       ↑
    阅读（10%记住）
       ↓
被动输入（最容易忘）

→ 越主动，记住越多
→ 越被动，忘得越快
```

## 3. 执行步骤

### Step 1: 分析考试

**知己知彼**

```
1. 考试考什么？
   - 考试大纲/历年真题
   - 重点章节
   - 题型分布

2. 多少分及格？
   - 及格线是多少
   - 重点抓分章节

3. 什么时候考？
   - 倒推时间
   - 制定计划
```

### Step 2: 评估现状

```
1. 现在什么水平？
   - 做一套真题自测
   - 算出分数分布

2. 哪些会？哪些不会？
   - 会的：巩固，确保不失分
   - 不会的：重点突破

3. 差多少分能及格/上岸？
   - 目标分数 - 当前分数 = 需要提升
```

### Step 3: 制定计划

** SMART 原则 **

```
S - Specific（具体）
   → 不是"复习数学"
   → 而是"今天学完函数专题"

M - Measurable（可衡量）
   → 不是"多做点题"
   → 而是"完成50道函数题"

A - Achievable（可实现）
   → 不是"一天学10小时"
   → 而是"每天高效4小时"

R - Relevant（相关）
   → 学的都是要考的

T - Time-bound（有时限）
   → 规定时间内完成
```

### Step 4: 高效执行

```
每日流程（以4小时为例）：

第一小时：复习旧知识
- 回顾昨天内容
- 复习笔记

第二小时：学习新知识
- 看教材/视频
- 理解概念

第三小时：做题巩固
- 做相关练习题
- 错题记录

第四小时：总结输出
- 整理笔记
- 自我检测
```

## 4. 各科复习策略

### 4.1 语文

```
【作文】
- 每周写1-2篇
- 背20篇好范文
- 积累好词好句

【阅读】
- 掌握答题模板
- 每天做1篇阅读

【古文】
- 背120个实词
- 每天读一篇古文

【基础】
- 字音字形：每天练
- 成语病句：每天练
```

### 4.2 数学

```
【基础题（选择填空）】
- 确保正确率95%以上
- 限时训练
- 错题本

【大题】
- 三角函数：公式记熟
- 立体几何：画图能力
- 概率统计：理解概念
- 解析几何：套路练习
- 导数：第一问要会

策略：
- 大题第一问一定要做完
- 不会也要写步骤拿分
```

### 4.3 英语

```
【单词】
- 每天50个
- 用艾宾浩斯曲线复习
- 放在句子里背

【阅读】
- 每天2篇
- 积累生词
- 分析长难句

【听力】
- 每天听20分钟
- 先看题再听
- 听写训练

【作文】
- 背10篇范文
- 用高级句型
- 字迹工整
```

### 4.4 理综（物理/化学/生物）

```
【物理】
- 理解公式推导
- 每个公式要知道怎么来的
- 做物理题要画图

【化学】
- 元素周期表要熟
- 方程式要配平
- 实验题背步骤

【生物】
- 课本是核心
- 背课本原话
- 遗传题多练
```

### 4.5 文综（历史/地理/政治）

```
【历史】
- 时间线要清楚
- 背重要事件
- 理解背景、过程、影响

【地理】
- 自然地理要理解（大气、洋流）
- 人文地理要记忆（区位因素）
- 地图要熟悉

【政治】
- 背关键词
- 答题分点
- 关注时事
```

## 5. 提分技巧

### 5.1 错题本方法

```
错题本格式：

日期：2024/1/1
错题：第X题
题目：xxxxx
错误答案：xxxxx
正确答案：xxxxx
原因：xxxxx
类型：计算错误/概念不清/审题失误

每周复习一次错题本
每次复习打勾表示掌握
```

### 5.2 考前一周

```
倒数第7天：巩固重点
- 最重要的知识点
- 错题本过一遍

倒数第6天：做模拟题
- 限时做一套
- 找到薄弱点

倒数第5天：补薄弱点
- 专项练习薄弱章节

倒数第4天：做真题
- 做近几年真题
- 分析出题规律

倒数第3天：轻松复习
- 看笔记
- 不做新题

倒数第2天：放松
- 适当运动
- 早睡

考试当天：带齐东西
- 证件
- 文具
- 充满电的计算器
```

### 5.3 考试技巧

```
【时间分配】
- 选择填空：X分钟
- 大题：Y分钟
- 检查：Z分钟
- 到点就过，不纠结

【做题顺序】
- 先做会的
- 后做不会的
- 确保会的都对

【选择题技巧】
- 排除法
- 代入验证
- 特殊值法

【大题技巧】
- 不会也要写步骤
- 公式写上去
- 单位要写对

【检查重点】
- 计算错误
- 审题失误
- 漏做题目
```

## 6. 心理调节

### 6.1 考前紧张

```
正常反应：
- 紧张是正常的
- 适度紧张有助于发挥

缓解方法：
1. 深呼吸（4秒吸气，8秒呼气）
2. 适当运动
3. 正常作息
4. 自我暗示
```

### 6.2 考试中慌了

```
方法1：暂停
- 停下来，深呼吸
- 告诉自己没关系

方法2：做简单题
- 先做会的
- 找回信心

方法3：心理暗示
- 大家都一样难
- 我不会的别人也不会
```

### 6.3 考后调整

```
考完一科，忘掉一科
- 不要对答案
- 专注下一科

考完所有：
- 总结经验教训
- 好的继续保持
- 坏的下次改进
```

## 7. 常见误区

### ❌ 误区1：复习=看书

**✅ 正确：**
- 要输出，要做题
- 看书是输入，做题是输出

### ❌ 误区2：时间花得越长越好

**✅ 正确：**
- 效率比时间重要
- 累了就休息
- 4小时高效 > 8小时低效

### ❌ 误区3：题海战术就能提分

**✅ 正确：**
- 做完要总结
- 同类题做透就行
- 不在于做多少，在于掌握多少

### ❌ 误区4：只学自己喜欢的科目

**✅ 正确：**
- 短板决定总分
- 薄弱科目提分空间大
