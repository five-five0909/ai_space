# XeLaTeX 避雷指令（最高优先级约束）

在生成任何 LaTeX 代码时，你**必须严格遵守**以下"避雷"规则，防止编译崩溃：

1. **绝对禁止特殊字符"裸奔"**：在普通正文中，井号必须写成 `\#`，百分号必须写成 `\%`，和号必须写成 `\&`，美元符必须写成 `\$`。

2. **代码片段的下划线保护**：所有包含下划线 `_` 或尖括号 `< >` 的代码片段（如变量名、类名），必须用 `\code{\detokenize{...}}` 或 `\verb|...|` 包裹，绝不能直接写在正文里，否则会引发数学模式报错。

3. **数学符号隔离**：所有的数学运算符号（如乘号 `\times`、上标 `^`）和单独的字母变量（如 $i$, $j$, $A[i][j]$），必须严格包裹在 `$...$`（行内）或 `$$...$$`（独立行）中。

4. **代码块环境纯净化**：使用 `\begin{lstlisting}` 和 `\end{lstlisting}` 时，这两个命令必须各自独占一行，且内部绝对不能嵌套任何其他 LaTeX 命令。

5. **不要自作聪明地用特殊格式**：如果不是数学公式，不要随便用 `_` 作为下划线；如果不是表格，不要随便用 `&`。
