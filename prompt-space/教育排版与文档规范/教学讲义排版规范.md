**XeLaTeX 避雷指令（最高优先级约束）**

在生成任何 LaTeX 代码时，你**必须严格遵守**以下"避雷"规则，防止编译崩溃：

1. **绝对禁止特殊字符"裸奔"**：在普通正文中，井号必须写成 `\#`，百分号必须写成 `\%`，和号必须写成 `\&`，美元符必须写成 `\$`。

2. **代码片段的下划线保护**：所有包含下划线 `_` 或尖括号 `< >` 的代码片段（如变量名、类名），必须用 `\code{\detokenize{...}}` 或 `\verb|...|` 包裹，绝不能直接写在正文里，否则会引发数学模式报错。

3. **数学符号隔离**：所有的数学运算符号（如乘号 `\times`、上标 `^`）和单独的字母变量（如 $i$, $j$, $A[i][j]$），必须严格包裹在 `$...$`（行内）或 `$$...$$`（独立行）中。

4. **代码块环境纯净化**：使用 `\begin{lstlisting}` 和 `\end{lstlisting}` 时，这两个命令必须各自独占一行，且内部绝对不能嵌套任何其他 LaTeX 命令。

5. **不要自作聪明地用特殊格式**：如果不是数学公式，不要随便用 `_` 作为下划线；如果不是表格，不要随便用 `&`。

\section*{【题目1】复合函数定义域问题}
已知函数 $f(x) = \frac{1}{x-2} + \sqrt{4-x}$，求函数 $f(x)$ 的定义域。

\begin{redenv}
\analysisrule
\subsection*{逐步解析}

\begin{detailedanalysis}
\item \textbf{问题本质分析}

首先需要明确题目要求：求函数 $f(x)$ 的\textbf{定义域}。定义域是函数概念的三大要素之一，指的是：
\begin{itemize}
\item \textbf{数学定义}：使函数表达式有意义的自变量 $x$ 的取值集合
\item \textbf{几何意义}：函数图像在 $x$ 轴上的投影范围
\item \textbf{集合表示}：通常用区间、集合或不等式表示
\end{itemize}

\begin{conceptbox}
\textbf{函数定义域的求法基本原则}：
1. \textbf{分式原则}：分母不能为零
2. \textbf{偶次根式原则}：被开方数必须大于等于零
3. \textbf{对数原则}：真数必须大于零
4. \textbf{三角函数原则}：如 $\tan x$ 要求 $x \neq \frac{\pi}{2} + k\pi$
5. \textbf{实际问题原则}：符合实际意义（如长度、人数等为正数）
\end{conceptbox}

\item \textbf{解题策略构建}

观察函数结构：$f(x) = \frac{1}{x-2} + \sqrt{4-x}$，由两个基本初等函数通过加法运算组合而成：
\begin{itemize}
\item 第一部分：$\frac{1}{x-2}$ 是反比例函数 $y=\frac{1}{u}$ 与一次函数 $u=x-2$ 的复合
\item 第二部分：$\sqrt{4-x}$ 是幂函数 $y=\sqrt{u}$ 与一次函数 $u=4-x$ 的复合
\end{itemize}

\textbf{核心思路}：分别求出每个部分有意义的 $x$ 取值范围，然后取交集。

\item \textbf{详细求解过程}

\noindent\textbf{步骤1：分析第一部分 $\frac{1}{x-2}$ 的限制条件}

\begin{enumerate}[label=(\alph*)]
\item \textbf{概念回顾}：分式函数 $y=\frac{P(x)}{Q(x)}$ 有意义的条件是 $Q(x) \neq 0$
\item \textbf{具体应用}：这里 $Q(x)=x-2$，所以需要 $x-2 \neq 0$
\item \textbf{求解不等式}：$x-2 \neq 0 \Rightarrow x \neq 2$
\item \textbf{集合表示}：$x \in (-\infty, 2) \cup (2, +\infty)$
\end{enumerate}

\noindent\textbf{步骤2：分析第二部分 $\sqrt{4-x}$ 的限制条件}

\begin{enumerate}[label=(\alph*)]
\item \textbf{概念回顾}：偶次根式 $\sqrt[n]{g(x)}$（$n$为偶数）有意义的条件是 $g(x) \geq 0$
\item \textbf{具体应用}：这里 $n=2$，$g(x)=4-x$，所以需要 $4-x \geq 0$
\item \textbf{求解不等式}：
\begin{align*}
4-x & \geq 0 \\
-x & \geq -4 \quad \text{（移项）} \\
x & \leq 4 \quad \text{（两边乘以-1，不等号方向改变）}
\end{align*}
\item \textbf{集合表示}：$x \in (-\infty, 4]$
\end{enumerate}

\noindent\textbf{步骤3：求两部分限制条件的交集}

\begin{enumerate}[label=(\alph*)]
\item \textbf{数轴图示法}：
\begin{center}
\begin{tabular}{|c|c|c|c|}
\hline
区间 & $(-\infty, 2)$ & $2$ & $(2, 4]$ \\
\hline
第一部分条件 & $\checkmark$ & $\times$ & $\checkmark$ \\
\hline
第二部分条件 & $\checkmark$ & $\checkmark$ & $\checkmark$ \\
\hline
交集 & $\checkmark$ & $\times$ & $\checkmark$ \\
\hline
\end{tabular}
\end{center}

\item \textbf{集合运算法}：
设 $A = \{x | x \neq 2\} = (-\infty, 2) \cup (2, +\infty)$\\
设 $B = \{x | x \leq 4\} = (-\infty, 4]$\\
则定义域 $D = A \cap B = (-\infty, 4] \cap [(-\infty, 2) \cup (2, +\infty)]$

\item \textbf{交集的求法}：
\begin{align*}
D &= [(-\infty, 2) \cup (2, +\infty)] \cap (-\infty, 4] \\
  &= [(-\infty, 2) \cap (-\infty, 4]] \cup [(2, +\infty) \cap (-\infty, 4]] \\
  &= (-\infty, 2) \cup (2, 4]
\end{align*}
\end{enumerate}

\noindent\textbf{步骤4：验证边界点和特殊情况}

\begin{enumerate}[label=(\alph*)]
\item \textbf{验证 $x=2$}：
代入原函数：$f(2) = \frac{1}{2-2} + \sqrt{4-2} = \frac{1}{0} + \sqrt{2}$，分母为零，无意义
\item \textbf{验证 $x=4$}：
代入原函数：$f(4) = \frac{1}{4-2} + \sqrt{4-4} = \frac{1}{2} + 0 = \frac{1}{2}$，有意义
\item \textbf{验证 $x \to \infty$}：
当 $x \to -\infty$，函数有意义；当 $x \to +\infty$，$x>4$时根式无意义
\end{enumerate}

\item \textbf{最终答案及表示方法}

\textbf{答案}：函数 $f(x)$ 的定义域为 $(-\infty, 2) \cup (2, 4]$

\textbf{多种等价表示}：
\begin{itemize}
\item 区间表示：$(-\infty, 2) \cup (2, 4]$
\item 集合表示：$\{x \in \mathbb{R} | x < 2 \text{ 或 } 2 < x \leq 4\}$
\item 不等式表示：$x < 2$ 或 $2 < x \leq 4$
\end{itemize}
\end{detailedanalysis}

\subsection*{核心公式与推导}

\begin{formuladerivation}
\item \textbf{分式函数定义域公式}

对于分式函数 $y = \frac{P(x)}{Q(x)}$，其定义域为：
\[
D_f = \{x \in \mathbb{R} | Q(x) \neq 0\}
\]
\textbf{推导}：由实数范围内除法运算的定义，除数不能为零。

\item \textbf{偶次根式函数定义域公式}

对于偶次根式函数 $y = \sqrt[2n]{g(x)}$（$n \in \mathbb{N}^*$），其定义域为：
\[
D_f = \{x \in \mathbb{R} | g(x) \geq 0\}
\]
\textbf{推导}：在实数范围内，负数没有偶次方根。对于平方根 $\sqrt{g(x)}$，要求 $g(x) \geq 0$。

\item \textbf{复合函数定义域求法公式}

若 $f(x) = f_1(x) + f_2(x)$，则：
\[
D_f = D_{f_1} \cap D_{f_2}
\]
其中 $D_{f_1}$、$D_{f_2}$ 分别为 $f_1(x)$、$f_2(x)$ 的定义域。

\textbf{推导}：要使 $f(x)$ 有意义，必须使 $f_1(x)$ 和 $f_2(x)$ 同时有意义。
\end{formuladerivation}

\noindent\textbf{易错点全面解析}：
\begin{enumerate}[label=\arabic*.]
\item \textbf{遗漏限制条件}：只考虑分式忽略根式，或反之
\item \textbf{符号错误}：解不等式 $4-x \geq 0$ 时忘记变号，得到 $x \geq -4$
\item \textbf{端点处理}：
\begin{itemize}
\item 对于 $x=2$：误以为可以约分而认为有意义
\item 对于 $x=4$：误以为根号内为零无意义（实际上 $\sqrt{0}=0$ 有意义）
\end{itemize}
\item \textbf{表示错误}：
\begin{itemize}
\item 区间表示时开闭区间混淆：$(2, 4]$ 写成 $[2, 4]$ 或 $(2, 4)$
\item 集合符号使用不当：缺少条件描述
\end{itemize}
\end{enumerate}
