# 前端迁移指南：JSP → Vue + ElementUI

---

## 一、迁移概述

### 1.1 项目信息

| 项目 | 原技术栈 | 目标技术栈 |
|------|---------|-----------|
| 新闻管理系统前端 | JSP + jQuery + Bootstrap | Vue 3 + Element Plus + Axios |

### 1.2 迁移范围

- **前台页面**: 新闻列表、新闻详情、分类浏览、评论功能
- **后台页面**: 登录、新闻管理、分类管理、评论管理、链接管理
- **API对接**: 调用新的 Spring Boot REST API

### 1.3 目标技术栈

| 组件 | 版本/说明 |
|------|-----------|
| Node.js | >= 18.0.0 |
| Vue | 3.x |
| Vite | 5.x (构建工具) |
| Vue Router | 4.x |
| Pinia | 2.x (状态管理) |
| Element Plus | 2.x (UI组件库) |
| Axios | 1.x (HTTP客户端) |
| CKEditor 5 | 富文本编辑器 |

#### 4.3.3 utils/request.js Axios封装

```javascript
import axios from 'axios'
import { ElMessage } from 'element-plus'

const request = axios.create({
  baseURL: import.meta.env.VITE_API_BASE_URL || 'http://localhost:8080',
  timeout: 15000,
  headers: {
    'Content-Type': 'application/json'
  }
})

// 请求拦截器
request.interceptors.request.use(
  (config) => {
    const token = localStorage.getItem('token')
    if (token) {
      config.headers.Authorization = `Bearer ${token}`
    }
    return config
  },
  (error) => {
    return Promise.reject(error)
  }
)

// 响应拦截器
request.interceptors.response.use(
  (response) => {
    const res = response.data
    if (res.code === 200) {
      return res
    } else {
      ElMessage.error(res.msg || '请求失败')
      return Promise.reject(new Error(res.msg || '请求失败'))
    }
  },
  (error) => {
    ElMessage.error(error.message || '网络错误')
    return Promise.reject(error)
  }
)

export default request
```

#### 4.5.2 前台新闻列表 (views/front/news/List.vue)

```vue
<template>
  <FrontLayout>
    <div class="news-list">
      <h2>{{ currentTypeName || '新闻列表' }}</h2>

      <div class="filter-bar">
        <el-input
          v-model="searchKey"
          placeholder="搜索新闻"
          clearable
          @clear="handleSearch"
          style="width: 200px"
        />
        <el-button type="primary" @click="handleSearch">搜索</el-button>
      </div>

      <div class="news-items">
        <div
          v-for="news in newsList"
          :key="news.newsId"
          class="news-item"
          @click="goToDetail(news.newsId)"
        >
          <h3>{{ news.title }}</h3>
          <div class="meta">
            <span>{{ formatDate(news.publishDate) }}</span>
            <span>作者：{{ news.author }}</span>
            <span>点击：{{ news.click }}</span>
          </div>
        </div>
      </div>

      <el-pagination
        v-model:current-page="page"
        :page-size="pageSize"
        :total="total"
        layout="prev, pager, next"
        @current-change="handlePageChange"
      />
    </div>
  </FrontLayout>
</template>

<script setup>
import { ref, computed, onMounted, watch } from 'vue'
import { useRoute, useRouter } from 'vue-router'
import { getNewsList } from '@/api/news'
import { formatDate } from '@/utils/date'
import FrontLayout from '@/layouts/FrontLayout.vue'

const route = useRoute()
const router = useRouter()

const newsList = ref([])
const total = ref(0)
const page = ref(1)
const pageSize = ref(10)
const searchKey = ref('')
const typeId = computed(() => route.query.typeId)

const currentTypeName = computed(() => {
  // 可以从路由或状态中获取
  return ''
})

const fetchNewsList = async () => {
  const res = await getNewsList({
    typeId: typeId.value,
    page: page.value
  })
  const data = res.data
  newsList.value = data.newsList
  total.value = data.total
}

onMounted(fetchNewsList)

watch(typeId, fetchNewsList)

const handleSearch = () => {
  page.value = 1
  fetchNewsList()
}

const handlePageChange = (newPage) => {
  page.value = newPage
  fetchNewsList()
}

const goToDetail = (newsId) => {
  router.push(`/news/${newsId}`)
}
</script>
```
