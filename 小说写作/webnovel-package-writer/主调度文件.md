# 网文写作 Skill · 主调度文件 v5.0

## 系统说明

本系统所有规则存放于 `规则/` 目录下。
每次执行任务前，根据任务类型自动加载对应规则文件。
规则文件扩展名为 `.rule`，每次使用前必须读取对应文件内容。

**重要：本系统采用标准化目录结构，所有小说项目必须放入 `书籍/` 目录。**
**详见：标准化规范.md**

---

## 规则文件索引

| 规则文件 | 路径 | 加载时机 |
|----------|------|----------|
| 语言规范 | 规则/language.rule | 每次生成正文前 |
| 剧情结构 | 规则/plot.rule | 写正文和大纲时 |
| 人设感情 | 规则/character.rule | 写人物内容时 |
| 检查规范 | 规则/review.rule | 检查大纲正文时 |
| 输出格式 | 规则/output.rule | 每次输出文件前 |
| 字数规范 | 规则/wordcount.rule | 每次输出文件前 |
| 空行清理 | 规则/cleanup.rule | 每次保存文件后 |

## 核心风格要求

- **文风直白洗练** - 多用短句和直接的动作/对话描写
- **爽文节奏** - 爽点前置，不拖沓
- **杀伐果断** - 主角行为利落，不圣母
- **降维打击感** - 主角对原著人物有绝对压制力
- **信息密度高** - 每句都有信息量，拒绝注水

## 输出格式规范

### 章节文件命名
```
✓ 正确：第 X 章 章节名.txt
✗ 错误：chapter-001.txt
✗ 错误：第 01 章 穿越.txt
✗ 错误：第 1 章.txt（无章节名）
```

### 章节名规范
- 字数：2-8 字
- 内容：本章最核心事件/转折
- 风格：简洁有力，有画面感
- 禁止：剧透结局、使用疑问句、超过 8 字

### 字数规范
- 每章正文：3800-4000 字
- 低于 3800 字：补写至达标
- 超过 4000 字：裁剪至达标

### 格式规范
- 文件格式：纯文本 .txt
- 禁止 Markdown 字符（# ** * 等）
- 无空行（cleanup.rule 自动清理）
