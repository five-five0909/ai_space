## 一、世界观核心规则

### 体系融合设定
- **查克拉本质**：查克拉 = 修仙界最原始的灵气，低级但兼容性高
- **仙术破解**：
  - 原著缺陷：需静止吸能、易石化、门槛高
  - 主角方案：炼体抗石化 + 御灵边打边吸

### 宗门设定：陌道宗
- **规矩**：只有一条——"进了门，生死我担"
- **信物**：腰牌刻字"陌道无界"
- **地位**：从最初的小黑屋，发展为凌驾五影之上的忍界第三股势力

## 四、陌尘完整人设卡

| 项目 | 内容 |
|------|------|
| **姓名** | 陌尘（火影世界户籍名，自取） |
| **前世称号** | 陌道尊（修仙界魔道第一人） |
| **穿越原因** | 寿元将尽冲击最高境界，肉身崩解时撞上波风水门的空间波动，残魂穿界 |
| **火影起点** | 木叶孤儿院，与鸣人同龄 |
| **年龄** | 12岁（故事开始时） |
| **外貌** | 黑发黑瞳，常年面无表情，气质冷硬，眼神有天然威压（神识外放） |

### 性格画像
- 冷：面无表情，话少
- 懒：不主动找事
- 护短：收徒即护短
- 绝对理性：以利益判断为主

### 行事原则
1. 不主动害无辜
2. 动我的人必死
3. 从不解释

### 弱点
对"被抛弃/被背叛"极度敏感，对徒弟唯一要求是"忠诚"，最恨背叛。

### 口头禅
- 「够了。」
- 「不值得。」
- 「随便。」
- 「无聊。」

### 从正文第11-15章提取的语言风格特征
- 多用"嗯"代替完整回答
- 不解释自己的行为
- 直接行动，少说话
- 对无聊的事情不屑一顾
- 站着不动等对方攻击
- 一招结束战斗
- 转身就走
