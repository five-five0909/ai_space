## 一、世界观核心规则

### 现实设定
```
异变前：24岁，普通便利店夜班店员，无背景无靠山
    ↓ 第33次副本触发时突然觉醒
现在：便利店副本固定NPC「深夜店员」，唯一有跨副本记忆的异常个体
    ↓ 系统上线
目标：从被猎杀的异常NPC，成为规则世界新的制定者
```

### 世界现状
| 项目 | 内容 |
|------|------|
| **现实时间** | 当代中国，异变发生在某年某月某日凌晨0点，全球同步 |
| **异变表现** | 城市里开始出现"封锁区"，普通人进入后触发副本，不完成规则就无法出来 |
| **副本密度** | 平均每个城市每天触发3-5个副本，触发地点随机 |
| **高频场景** | 便利店、医院、学校、地铁 |
| **NPC认知** | 所有玩家都认为NPC是系统生成的工具人，没有意识，可以随意利用或消耗 |

### 规则世界铁律【补充】
1. 进入副本必须遵守规则
2. 违反规则直接抹杀
3. 完成规则才能离开
4. NPC是系统工具人（玩家认知）
5. 觉醒NPC是被清除对象（系统设定）

## 五、陆零完整人设卡

| 项目 | 内容 |
|------|------|
| **姓名** | 陆零（谐音"归零"，暗示从NPC重新开始） |
| **年龄** | 24岁 |
| **异变前身份** | 便利店夜班店员，大专毕业，家庭普通，无特殊背景 |
| **副本身份** | 「深夜店员」，便利店副本固定NPC，玩家攻略评级：无害，可无视 |
| **外貌** | 普通，不起眼，永远穿便利店工作服 |

### 性格核心
表面平静，内心极度清醒，不冲动，每次出手前已想好三步后的结果。

### 行事原则
1. 不主动暴露能力
2. 能用规则解决就不动手
3. 救过我的人我记着

### 弱点设计
过度理性，早期不相信任何人，错过几次更快成长的机会。

### 口头禅
- "规则里没说不行。"
- "你读过规则吗？"
- "我看过。"

### 最强武器
不是战斗力，是对规则的理解比任何玩家和管理者都深。
