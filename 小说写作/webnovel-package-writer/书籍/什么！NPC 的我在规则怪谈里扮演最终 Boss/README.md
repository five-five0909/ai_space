## 详细简介
陆零是个 NPC。

准确地说，他是便利店副本的夜班店员 NPC，每天的工作就是站在收银台后面，对进来的玩家说"欢迎光临"，然后看着他们一个接一个地死在规则下面。

他本该是个没有意识的工具人。

直到第三十三次循环，他觉醒了。

系统上线，金手指到账——【规则编辑器】。

他能看到每一条规则的漏洞，能溯源每一条规则的创建者，甚至能修改规则本身。

但代价是：每用一次能力，他的"存在值"就会下降。降到零，他就会变成真正的 NPC，永远消失。

更麻烦的是——他在追查规则来源时发现，规则的创建者，名字里也有一个"零"字。

和他同名。

## 核心设定

### 金手指：规则编辑器
| 层级 | 功能 | 说明 |
|------|------|------|
| 第一层 | 规则阅读 + 漏洞标注 | 看见完整规则含隐藏条款 |
| 第二层 | 规则注释 + 小幅修改 | 消耗积分修改单条规则 |
| 第三层 | 规则创造+Boss 模式 | 自己写规则插入副本 |
