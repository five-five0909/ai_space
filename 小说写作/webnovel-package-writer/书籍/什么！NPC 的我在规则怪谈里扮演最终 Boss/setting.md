## 一、世界观核心规则

### 现实设定
```
异变前：24岁，普通便利店夜班店员，无背景无靠山
    ↓ 第33次副本触发时突然觉醒
现在：便利店副本固定NPC「深夜店员」，唯一有跨副本记忆的异常个体
    ↓ 系统上线
目标：从被猎杀的异常NPC，成为规则世界新的制定者
```

### 世界现状
| 项目 | 内容 |
|------|------|
| **现实时间** | 当代中国，异变发生在某年某月某日凌晨0点，全球同步 |
| **异变表现** | 城市里开始出现"封锁区"，普通人进入后触发副本，不完成规则就无法出来 |
| **副本密度** | 平均每个城市每天触发3-5个副本，触发地点随机 |
| **高频场景** | 便利店、医院、学校、地铁 |
| **NPC认知** | 所有玩家都认为NPC是系统生成的工具人，没有意识，可以随意利用或消耗 |

### 规则世界铁律【补充】
1. 进入副本必须遵守规则
2. 违反规则直接抹杀
3. 完成规则才能离开
4. NPC是系统工具人（玩家认知）
5. 觉醒NPC是被清除对象（系统设定）

### 【优化补充】规则编辑器隐藏功能

在原有六大功能基础上追加：

#### 规则溯源
- **功能**：查看某条规则的创建者和创建时间
- **解锁条件**：累计修改规则超过100次
- **关键剧情**：解锁时主角看见便利店那条规则的创建者一栏是乱码，这是全书第一个真正的悬念

#### 规则共鸣
- **功能**：感知其他副本中被修改过的规则留下的痕迹
- **解锁条件**：NPC联盟建立后
- **用途**：用于找到其他觉醒NPC

#### 规则否决
- **功能**：直接否定某条规则的执行，无需修改，但消耗极大，使用后主角会短暂失去规则编辑器的所有功能
- **解锁条件**：规则创造功能解锁后
