## 详细简介

陆零是个NPC。

准确地说，他是便利店副本的夜班店员NPC，每天的工作就是站在收银台后面，对进来的玩家说"欢迎光临"，然后看着他们一个接一个地死在规则下面。

他本该是个没有意识的工具人。

直到第三十三次循环，他觉醒了。

系统上线，金手指到账——【规则编辑器】。

他能看到每一条规则的漏洞，能溯源每一条规则的创建者，甚至能修改规则本身。

但代价是：每用一次能力，他的"存在值"就会下降。降到零，他就会变成真正的NPC，永远消失。

更麻烦的是——他在追查规则来源时发现，规则的创建者，名字里也有一个"零"字。

和他同名。

裁决者告诉他：三年前你不是变成NPC，三年前你就是规则的创建者。

他想知道真相。

但规则世界的管理者想要他死。

猎杀觉醒NPC的规则客正在逼近。

他必须做出选择——

躲起来，等待消失。

还是撕碎规则，成为这个世界的最终Boss。

## 核心卖点

### 1. 身份反转爽
从最底层的NPC工具人，一路成长为规则世界的创造者与统治者。每次身份揭露都是一次爽点爆发。

### 2. 规则智斗爽
主角不是靠武力碾压，而是靠对规则的理解比别人深三层。敌人以为规则对自己有利，主角早就改了。

### 3. NPC觉醒爽
帮助其他NPC觉醒，建立NPC联盟，从被猎杀者变成猎杀者，整个群体的逆袭。

### 4. 信息差碾压
唯一拥有跨副本记忆的存在，知道别人不知道的信息，利用信息差实现降维打击。

### 5. 韦小宝式主角
表面滑头爱占便宜，实际有原则不卖队友。嘴上说"我就是手欠"，行动上把所有人护在身后。

## 核心设定

### 金手指：【规则编辑器】

| 功能 | 说明 |
|------|------|
| 规则阅读 | 查看副本所有规则（包括隐藏规则） |
| 漏洞标注 | 自动标注规则漏洞和逻辑缺陷 |
| 规则溯源 | 追查规则的创建者和创建时间 |
| 规则修改 | 修改副本规则（消耗存在值） |
| 规则共鸣 | 与其他觉醒NPC建立联系 |
| 规则否决 | 第三层功能，可否决规则 |

### 代价机制

每次使用规则编辑器，存在值会下降：
- 存在值 < 50%：可主动申请成为副本Boss
- 存在值 = 0%：永久消失，变成真正的无意识NPC
