陌道尊，修仙界魔道第一人，寿元将尽冲击最高境界时肉身崩解，残魂撞上波风水门的空间波动，穿越到火影世界，成为木叶孤儿院中与鸣人同龄的孤儿。

带着前世《混沌魔典》和储物戒资源，陌尘在忍者世界普化仙道。

收鸣人为大弟子，建陌道宗凌驾五影之上，改写所有意难平。

修仙体系碾压忍术体系，降维打击整个忍界。
