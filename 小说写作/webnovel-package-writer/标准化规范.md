# 网文创作系统标准化规范 v1.0（全中文命名版）

## 一、目录结构规范

```
webnovel-package-writer/
├── 主调度文件.md             # 主调度文件（必须）
├── 标准化规范.md             # 本规范文件（必须）
│
├── 规则/                     # 规则文件目录（必须，不可删除）
│   ├── language.rule        # 语言规范
│   ├── plot.rule            # 剧情结构
│   ├── character.rule       # 人设感情
│   ├── review.rule          # 检查规范
│   ├── output.rule          # 输出格式
│   ├── wordcount.rule       # 字数规范
│   └── cleanup.rule         # 空行清理
│
├── 模块/                     # 功能模块（必须，不可删除）
│
├── 书籍/                     # 【所有小说项目存放处】
│   ├── 火影同人/             # 小说 1：火影同人
│   │   ├── README.md        # 小说主文件（必须）
│   │   ├── 角色设定/        # 角色设定（可选）
│   │   ├── 剧情大纲/        # 剧情大纲（可选）
│   │   └── 章节产出/        # 章节产出（必须）
│   │
│   ├── 规则怪谈/             # 小说 2：规则怪谈
│   │   ├── README.md
│   │   ├── 角色设定/
│   │   ├── 剧情大纲/
│   │   └── 章节产出/
│   │
│   └── [新书名]/            # 新增小说按此模板
│       ├── README.md
│       ├── 角色设定/
│       ├── 剧情大纲/
│       └── 章节产出/
│
├── 素材/                     # 【公共资源目录】
│   └── 封面/                # 封面图片
│       ├── 火影同人-cover.png
│       └── 规则怪谈-cover.png
│
└── 资源库/                   # 通用素材库（必须，不可删除）
    ├── 通用素材/            # 两本书共用素材
    ├── 火影同人/            # 火影专属素材
    └── 规则怪谈/            # 规则怪谈专属素材
```

## 二、小说文件夹命名规范

### 目录命名规则
| 规则 | 说明 | 示例 |
|------|------|------|
| 语言 | 使用中文书名 | `火影同人`, `规则怪谈` |
| 禁止 | 英文、数字前缀 | ❌ `book1-` ❌ `naruto` |

### 正确示例
```
✓ 火影同人/
✓ 规则怪谈/
✓ 系统重生/
✓ 都市修仙/
```

### 错误示例
```
✗ book1-naruto/
✗ 01-火影/
✗ naruto/
```

## 三、小说内部结构规范

### 每本小说必须包含

```
[小说文件夹]/
├── README.md              # 【必须】小说主文件
│                          # 包含：书名、类型、标签、简介、核心设定
│
├── 角色设定/              # 【必须】角色设定目录
│   ├── 主角.md           # 主角人设卡
│   ├── 配角.md           # 配角人设卡
│   └── 反派.md           # 反派人设卡
│
├── 剧情大纲/              # 【必须】剧情大纲目录
│   ├── 全书大纲.md       # 全书大纲
│   ├── 章节大纲.md       # 章节大纲
│   └── 时间线.md         # 时间线（可选）
│
└── 章节产出/              # 【必须】章节产出目录
    ├── 第 1 章 章节名.txt
    ├── 第 2 章 章节名.txt
    └── ...
```

## 四、章节产出规范

### 文件命名
```
格式：第 X 章 章节名.txt
示例：第 1 章 穿越觉醒.txt
```

### 禁止项
```
✗ chapter-001.txt
✗ 第 01 章 穿越.txt
✗ 第 1 章.txt（无章节名）
✗ 第 1 章_穿越.txt（下划线）
```

## 六、主调度文件路径引用规范

### 书目路径格式
```markdown
## 书目资源路径对照

**火影同人：**
- 资源目录：资源库/火影同人/
- 产出路径：书籍/火影同人/章节产出/
- 封面路径：素材/封面/火影同人-cover.png

**规则怪谈：**
- 资源目录：资源库/规则怪谈/
- 产出路径：书籍/规则怪谈/章节产出/
- 封面路径：素材/封面/规则怪谈-cover.png
```

### 指令格式
```
【写正文·火影】第 X 章 + 章节要点
【写正文·规则】第 X 章 + 章节要点
【查人设·火影】角色名
【查大纲·规则】阶段
```

## 七、新增小说流程

### 步骤 1：创建文件夹
```
书籍/书名/
书籍/书名/角色设定/
书籍/书名/剧情大纲/
书籍/书名/章节产出/
```

### 步骤 2：创建 README.md

### 步骤 3：创建人设卡
```
书籍/书名/角色设定/主角.md
```

### 步骤 4：创建大纲
```
书籍/书名/剧情大纲/全书大纲.md
```

### 步骤 5：更新 主调度文件.md

### 步骤 6：创建专属资源目录（可选）
```
资源库/书名/
```

## 八、快速检查清单

新增小说前检查：
- [ ] 文件夹使用中文书名
- [ ] 包含 README.md
- [ ] 包含 角色设定/ 目录
- [ ] 包含 剧情大纲/ 目录
- [ ] 包含 章节产出/ 目录
- [ ] 封面已放入 素材/封面/
- [ ] 主调度文件.md 已更新路径

章节产出前检查：
- [ ] 文件名格式：第 X 章 章节名.txt
- [ ] 章节名 2-8 字
- [ ] 字数 3800-4000 字
- [ ] 无 Markdown 格式
- [ ] 无空行
