## 一、世界观核心规则

### 现实设定
```
异变前：24 岁，普通便利店夜班店员，无背景无靠山
    ↓ 第 33 次副本触发时突然觉醒
现在：便利店副本固定 NPC「深夜店员」，唯一有跨副本记忆的异常个体
    ↓ 系统上线
目标：从被猎杀的异常 NPC，成为规则世界新的制定者
```

### 世界现状
| 项目 | 内容 |
|------|------|
| **现实时间** | 当代中国，异变发生在某年某月某日凌晨 0 点，全球同步 |
| **异变表现** | 城市里开始出现"封锁区"，普通人进入后触发副本，不完成规则就无法出来 |
| **副本密度** | 平均每个城市每天触发 3-5 个副本，触发地点随机 |
| **高频场景** | 便利店、医院、学校、地铁 |
| **NPC 认知** | 所有玩家都认为 NPC 是系统生成的工具人，没有意识，可以随意利用或消耗 |

### 规则世界铁律
1. 进入副本必须遵守规则
2. 违反规则直接抹杀
3. 完成规则才能离开
4. NPC 是系统工具人（玩家认知）
5. 觉醒 NPC 是被清除对象（系统设定）

## 二、时代背景完整设定

### 异变起源
- 某年某月某日凌晨 0 点，全球同步异变
- 原因不明，无任何预兆
- 封锁区开始在城市中随机出现
- 政府知情但无法控制

### 社会变化
| 方面 | 变化 |
|------|------|
| 经济 | 攻略师成为高收入职业 |
| 政治 | 政府成立专门应对部门 |
| 社交 | 攻略社区成为主流社交平台 |
| 价值观 | 强者垄断资源，弱者沦为炮灰 |

## 三、副本等级体系

| 等级 | 类型 | 死亡率 | 说明 |
|------|------|--------|------|
| **F 级** | 单人副本 | 30% | 新手入门级 |
| **E 级** | 小队副本 | 50% | 需要基础配合 |
| **D 级** | 中型副本 | 70% | 需要一定攻略能力 |
| **C 级** | 大型副本 | 85% | 高难度，需要团队配合 |
| **B 级** | 城市级副本 | 95% | 极高难度，几乎没有生还者 |
| **A 级及以上** | 灾难级 | 已知无人生还 | 理论存在，无人通关 |

### 便利店副本评级
- **官方评级**：D 级
- **实际状况**：主角觉醒后死亡率变成 0%
- **系统标注**：异常副本，持续监测

## 四、玩家生态五类

| 类型 | 特点 | 代表人物 |
|------|------|----------|
| **散修玩家** | 单打独斗，死亡率最高 | 普通玩家 |
| **攻略组** | 靠信息差存活，林晚所属 | 林晚及团队 |

| **规则研究者** | 极少数，试图理解规则本质，全部神秘失踪 | 消失的学者们 |

## 五、NPC 觉醒现象设定

| 项目 | 内容 |
|------|------|
| **觉醒率** | 百万分之一 |
| **觉醒后名称** | "异常体" |
| **系统首要任务** | 规则执行者清除异常体 |
| **主角特殊标记** | 已知唯一存活超过 30 天的异常体 |
| **系统代号** | 【异常个体#0001】 |

### 觉醒阶段特征
| 阶段 | 表现 |
|------|------|
| 前兆期 | 出现短暂"卡顿"，行为超出程序设定 |
| 觉醒期 | 突然获得跨副本记忆，情绪崩溃或愤怒 |
| 稳定期 | 接受现实，开始寻求改变或自我毁灭 |

## 六、规则编辑器六大功能 + 三个隐藏功能

### 六大功能
| 层级 | 功能 | 具体内容 | 解锁时机 |
|------|------|----------|----------|
| 第一层 | 规则阅读 | 看见完整规则，包括隐藏条款 | 觉醒当天 |
| 第一层 | 漏洞标注 | 系统自动标注规则的逻辑漏洞 | 觉醒当天 |
| 第一层 | 漏洞分析 | 分析漏洞可利用的方式 | 觉醒当天 |
| 第二层 | 规则注释 | 给规则加备注，影响执行效果 | 中期解锁 |
| 第二层 | 小幅修改 | 消耗积分修改单条规则 | 中期解锁 |
| 第三层 | 规则创造 | 自己写规则插入副本 | 后期解锁 |
| 第三层 | Boss 模式 | 主动成为 Boss 获得能力加成 | 后期解锁 |

### 三个隐藏功能
| 功能 | 说明 | 解锁条件 |
|------|------|----------|
| 规则溯源 | 查看某条规则的创建者和创建时间 | 累计修改规则超过 100 次 |
| 规则共鸣 | 感知其他副本中被修改过的规则留下的痕迹 | NPC 联盟建立后 |
| 规则否决 | 直接否定某条规则的执行，无需修改 | 规则创造功能解锁后 |

### 积分系统
| 积分获取方式 | 数值 |
|------|------|
| 发现漏洞 | +10-50 分 |
| 利用漏洞成功 | +50-100 分 |
| 完成副本 | +100-500 分 |
| 帮助其他 NPC 觉醒 | +200 分 |
