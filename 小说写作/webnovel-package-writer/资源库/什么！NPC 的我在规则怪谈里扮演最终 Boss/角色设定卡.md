## 一、主角人设卡

### 【陆零】完整人设卡

| 项目 | 内容 |
|------|------|
| **姓名** | 陆零（谐音"归零"，暗示从 NPC 重新开始） |
| **年龄** | 24 岁 |
| **异变前身份** | 便利店夜班店员，大专毕业，家庭普通，无特殊背景 |
| **副本身份** | 「深夜店员」，便利店副本固定 NPC，玩家攻略评级：无害，可无视 |
| **外貌** | 普通，不起眼，永远穿便利店工作服 |

### 风格锚点定位
**韦小宝型**：滑头但有原则，从不害自己人，嘴上占便宜实际吃亏，看起来在划水实际上把所有人都算进去了。

### 性格四层结构

#### 第一层（表面）
- 能躺绝不站，能推绝不接
- 嘴上永远在抱怨、嫌麻烦、讲条件
- 对所有事表现出"关我什么事"

#### 第二层（实际）
- 嘴上说不管，已经悄悄解决了
- 嘴上说为了利益，其实是看不惯
- 事后必须找补一句：
  - "别误会，我就是顺路。"
  - "我只是手欠，下次不管了。"

#### 第三层（原则）
三件事绝对不做：
1. 不出卖自己人
2. 不对弱者动手
3. 不骗信任他的人

违反这三条的人，不管多强，他都会想办法搞死。

#### 第四层（真实）
- 比任何人都清楚规则世界的本质
- 从不说破，只是默默把人护在身后
- 独处时偶尔会看着手里的【规则编辑器】发呆
- 那是全书最少的几个安静时刻

### 行事原则
1. 不主动暴露能力
2. 能用规则解决就不动手
3. 救过我的人我记着

### 弱点设计
过度理性，早期不相信任何人，错过几次更快成长的机会。

### 口头禅体系（分场景）

#### 摸鱼划水时
- "这副本不合理，我要投诉。"
- "合着我是工具人是吧。"
- "行吧，但我有条件。"

#### 被人感谢时
- "行了行了别说了，烦死了。"

#### 看穿对方意图时
- "你的规则我看了，有三个漏洞。要我指出来吗？"
- （然后不等回答直接钻漏洞）

#### 真正生气时（全书不超过 5 次）
不说废话，沉默三秒，然后：
- "你动了不该动的人。"

#### 被问及动机时
- "我就是手欠，跟你没关系。"
- （是全书最大的谎言）

#### 原有口头禅保留
- "规则里没说不行。"
- "你读过规则吗？"
- "我看过。"

### 人设禁止项
| 禁止项 | 说明 |
|--------|------|
| ✗ 禁止主动承认自己在乎谁 | 永远用"顺路""手欠"等借口搪塞 |
| ✗ 禁止主动表白或说出"喜欢" | 感情只能通过行动体现 |
| ✗ 禁止在人多时表现出软的一面 | 示弱只能在背对人时 |
| ✗ 禁止被感动时让别人看见 | 可以背对着人，或用一句废话盖过去 |

### 最强武器
不是战斗力，是对规则的理解比任何玩家和管理者都深。

## 二、核心配角人设

### 【林晚】（一号·主攻略师）

| 项目 | 内容 |
|------|------|
| **身份** | 第一个被主角救的玩家，职业攻略师 |
| **外貌** | 干练短发，随身笔记本，走路比别人快半步 |
| **性格** | 表面精明强干，实际是全书第一个看穿主角不对劲的人，也是第一个选择闭嘴不说的人 |
| **与主角关系** | 从互相利用到唯一真正信任的搭档 |

#### 与主角关系发展线
| 阶段 | 内容 |
|------|------|
| 第一阶段 | 互相利用，她提供信息，他提供安全 |
| 第二阶段 | 拌嘴搭档，她用攻略思维分析他，越分析越看不懂 |
| 第三阶段 | 某次主角为她悄悄改了一条规则，她发现了但没说 |
| 第四阶段 | "我就是想跟着你，有问题吗？"主角沉默两秒："随便。"（这是他说过最温柔的话） |

#### 专属爽点
她整理的攻略，在主角手里全部失效，因为规则已经被悄悄改过了，她回头看他，他在看窗外。

#### 口头禅
- "按攻略你不应该这么做的。"
- "你这个 NPC 有问题。"
- （后期）"……算了，跟着你走就是了。"

### 【夜鸢】（二号·觉醒 NPC·游乐场副本）

| 项目 | 内容 |
|------|------|
| **身份** | 游乐场副本觉醒 NPC，售票员 |
| **外貌** | 永远戴着游乐场工作帽，帽檐压得有点低，笑起来有两个梨涡 |
| **性格核心** | 颠的，说话跳脱，思维跳跃让人跟不上，但每次最关键的时刻，她说的话是全场最准的 |

#### 与主角关系发展线
| 阶段 | 内容 |
|------|------|
| 第一次见面 | 就叫他"老板"。主角：我不是。夜鸢：你会是的。（她是第一个预判到主角终局的人） |
| 发展路径 | 从游乐场副本的困兽，到主角 NPC 联盟的情报核心 |

#### 专属爽点
在最危险的副本里，所有人绝望，她突然笑了："老板改过这里的规则，我刚感觉到。"然后局面反转。

#### 口头禅
- "好耶！"
- "这个副本有点好玩。"
- "老板你看我！"
- （关键时刻）沉默，然后轻声说："他在的，不会死的。"

## 四、肉体描写高密度规范

### 频率要求
每 3000 字必须有至少一个肢体细节描写，不能跳过，不能用对话代替。

### 高频使用细节库

#### 温度系列（每隔 5 章用一次）
- 体温透过衣料传过来
- 呼吸打在颈侧，有点烫
- 额头抵着额头能感觉到心跳
- 手心是热的

#### 触碰系列（每隔 3 章用一次）
- 手指碰到手背没有拿开
- 拉住手腕，力道比必要的大一点
- 指尖轻轻压了一下
- 发顶被按了一下

#### 距离系列（每隔 2 章用一次）
- 近到能看清睫毛
- 背靠背能感觉到呼吸节奏
- 肩膀已经靠在一起了
- 不知道什么时候手叠在一起了

### 写法规范
最多 3 句，必须立刻接回正文，禁止在细节上停留超过 50 字。
