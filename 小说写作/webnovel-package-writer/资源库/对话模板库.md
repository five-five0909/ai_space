# 对话模板库

> 各类型角色对话范例

---

## 一、主角对话模板

### 【陌尘】对话风格
- **特点**：话少、冷漠、不解释
- **常用回应**：嗯、无聊、秘密、够了、随便、不值得

#### 场景示例

**被问问题**
```
问：你为什么这么强？
陌尘：秘密。
或：嗯。（不回答）
```

**被挑衅**
```
对方：你一个小鬼，能有什么实力？
陌尘：你确定？
或：（直接出手）
```

**战斗结束**
```
敌人倒下。
陌尘：无聊。
或：太弱了。
```

**被感谢**
```
对方：谢谢你救了我！
陌尘：（点头离开）
或：任务。
```

**收徒**
```
对方：请收我为徒！
陌尘：不收。
（对方坚持后）
陌尘：好。但记住——进了门，生死我担。
```

---

### 【陆零】对话风格
- **特点**：理性、规则导向、不暴露
- **常用回应**：规则里没说不行、你读过规则吗、我看过、了解

#### 场景示例

**发现漏洞**
```
陆零：规则里没说不行。
或：这里有个漏洞。
```

**被质疑**
```
对方：你怎么知道的？
陆零：我看过。
或：规则写的。
```

**执行任务**
```
接受指令时：了解。
执行中：执行中。
被限制时：权限不足。
```

## 二、配角对话模板

### 热血型（鸣人）
```
"我一定会成为火影的！"
"说到做到，这就是我的忍道！"
"我不会放弃的！"
```

### 冷傲型（佐助）
```
"我要杀了那个人。"
"别挡路。"
"我不需要同伴。"
```

### 智囊型（鹿丸）
```
"真麻烦……"
"让我想想。"
"这招应该能行。"
```

### 疯狂型（大蛇丸）
```
"很有意思……"
"你的身体，我要了。"
"为了永生，一切都可以牺牲。"
```

### 忠诚型（白）
```
"再不斩大人的命令就是一切。"
"我愿意为再不斩大人付出生命。"
```

## 三、反派对话模板

### 嚣张型
```
"木叶的小鬼？"
"你确定要和我动手？"
"把东西交出来，我不杀你。"
```

### 阴险型
```
"呵呵，你以为这就结束了？"
"真是有趣的玩具。"
"慢慢享受死亡吧。"
```

### 狂妄型
```
"我是神！"
"蝼蚁也敢挑战我？"
"这个世界上没人能打败我！"
```

## 四、对话精简原则

### 删除项
| 删除 | 保留 |
|------|------|
| "你好，请问……" | 直接问 |
| "那个……我想问一下……" | 直接问 |
| "我想说的是……" | 直接说 |
| "我告诉你……" | 直接说 |

### 对话标签精简
| 原文 | 修改 |
|------|------|
| 他说道："……" | 他说："……" |
| 他开口说道："……" | 他："……" |
| 他回答道："……" | 他答："……" |
| "……"他说。 | "……"（省略标签） |

### 客套话删除
| 删除 | 效果 |
|------|------|
| "您好" | 直接说事 |
| "请问" | 直接问 |
| "谢谢" | 点头/离开 |
| "对不起" | 动作代替 |

## 五、对话节奏控制

### 快节奏（战斗/紧张）
```
"左边！"
"收到。"
"三个。"
"搞定。"
```

### 慢节奏（日常/铺垫）
```
"你说，他到底是什么人？"
卡卡西看了看窗外。
"不知道。但……有点意思。"
```
