# 爽点模板库

> 15种爽点模板，按类型调用

---

## 一、战力碾压类

### 模板1：一招秒杀
```
结构：主角不动 → 敌人攻击 → 主角一招 → 敌人倒下
示例：
- 陌尘站在原地。萨克的拳头轰过来。他一指点出。萨克飞了出去，撞在墙上，晕了。
要点：
- 主角不动显自信
- 一招结束不拖沓
- 结果直接呈现
```

### 模板2：碾压式对话
```
结构：敌人放狠话 → 主角无视/简短回应 → 一招秒杀 → 主角冷漠离开
示例：
- "你确定要我交卷轴？"陌尘看着他。
- 草隐村的中忍笑了。"你一个小鬼，能有什么实力？快交出来！"
- 陌尘叹了口气。"无聊。"
- 他抬手，一指点出。
要点：
- 对话极简
- 对比敌人话多vs主角话少
- 结尾留下"太弱了"的评价
```

## 二、智斗反转类

### 模板3：规则漏洞
```
结构：敌人设局 → 主角发现漏洞 → 利用漏洞反杀
示例：
- 敌人以为规则限制主角 → 主角发现"规则里没说不行" → 反杀
要点：
- 规则要清晰
- 漏洞要合理
- 反转要有冲击力
```

### 模板4：预判反制
```
结构：敌人出招 → 主角早有准备 → 揭示预判 → 敌人震惊
示例：
- 敌人使用禁术 → 主角拿出一物"三百年前见过这招" → 破解
要点：
- 展示主角见多识广
- 敌人震惊要到位
```

## 三、打脸类

### 模板5：被轻视后反杀
```
结构：被轻视 → 主角不解释 → 实力展示 → 众人震惊
示例：
- "木叶的小鬼？"萨克眼神轻蔑。
- 陌尘没说话。
- 一招后，萨克飞了出去。
- 全场安静。
要点：
- 轻视要明显
- 主角不辩解
- 结果要震撼
```

### 模板6：低调装逼
```
结构：低调表现 → 关键时刻出手 → 一鸣惊人
示例：
- 考试交白卷 → 被质疑 → 通过考试 → "我不需要证明什么"
要点：
- 前期低调
- 关键时刻反转
- 结尾金句
```

## 四、收徒类

### 模板7：拒绝后收服
```
结构：对方求收徒 → 主角拒绝 → 对方证明忠诚 → 收徒
示例：
- "收我为徒！" → "不收。" → 对方坚持/牺牲 → "好，入门。"
要点：
- 拒绝要坚决
- 证明要打动主角
- 收徒要有仪式感
```

### 模板8：点化式收徒
```
结构：对方困惑 → 主角点化一句话 → 对方顿悟 → 自动跟随
示例：
- "你的执念太弱了。为了一个利用你的人，不值得。"
- 白愣住了。
要点：
- 点化要精准
- 击中对方核心问题
```

## 五、反杀类

### 模板9：绝境翻盘
```
结构：陷入绝境 → 敌人得意 → 主角底牌 → 反杀
示例：
- 被围攻 → 敌人以为稳赢 → 主角使出隐藏实力 → 全灭
要点：
- 绝境要真实
- 底牌要合理
- 反杀要干脆
```

### 模板10：以弱胜强
```
结构：敌人强大 → 主角被发现弱点 → 一击致命
示例：
- 敌人是影级 → 主角发现其致命弱点 → 一指破防
要点：
- 弱点要有铺垫
- 击杀要精准
```

## 六、装逼类

### 模板11：淡定装逼
```
结构：众人紧张 → 主角淡定 → 一招解决 → 继续淡定
示例：
- 考生们都在紧张答题 → 陌尘交白卷 → 通过 → "无聊"
要点：
- 淡定对比紧张
- 结果出人意料
- 结尾保持人设
```

### 模板12：金句装逼
```
结构：关键时刻 → 主角说金句 → 一招制敌
示例：
- "你只是查克拉之祖。而我……是修仙者。"
要点：
- 金句要符合人设
- 时机要恰当
```

## 七、改变意难平类

### 模板13：救下必死之人
```
结构：原著必死场景 → 主角介入 → 救下 → 改变命运
示例：
- 宁次即将被杀 → 陌尘出现 → 挡下攻击 → "死人不行。活人可以。"
要点：
- 介入要合理
- 改变要有意义
```

### 模板14：揭露真相
```
结构：角色被误解 → 主角揭露真相 → 众人震惊
示例：
- 止水被怀疑 → 陌尘说"他的眼睛在团藏那里" → 真相大白
要点：
- 真相要有冲击力
- 时机要恰当
```

## 八、势力类

### 模板15：势力碾压
```
结构：敌人嚣张 → 提到主角势力 → 敌人害怕 → 臣服
示例：
- "你知道陌道宗吗？" → 敌人脸色变了 → 跪下
要点：
- 势力名要有威慑力
- 对比敌人前后态度
```

## 爽点使用频率规则

| 规则 | 说明 |
|------|------|
| 每章1个核心爽点 | 必须有 |
| 不连续3章同类型 | 保持新鲜感 |
| 每3章1个大爽点 | 收徒/反杀/改变意难平 |
| 每卷1个高潮爽点 | 势力碾压/最终对决 |
