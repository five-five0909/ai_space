# 词汇替换表

> 30+ 组 AI 烂词→人类化替换

---

## 一、连接词替换

| 原词 | 替换为 | 备注 |
|------|--------|------|
| 然后/接着 | 随即/转身/直接 | 或删除 |
| 突然之间 | 猛地/忽然 | 更短 |
| 不知不觉中 | 删除或替换 | AI 套话 |
| 转眼间 | 删除，直接跳转 | 废话过渡 |
| 时间一分一秒过去 | 删除，直接跳转 | 废话过渡 |

---

## 二、程度词替换

| 原词 | 替换为 | 备注 |
|------|--------|------|
| 非常/十分 | 极/格外 | 能删就删 |
| 特别/尤其 | 更是 | 精简 |
| 相当/颇为 | 很/挺 | 口语化 |
| 无比/极其 | 删除或具体描述 | 避免堆砌 |

| 走了过来 | 走过来/走过去 | 精简 |
| 坐了下来 | 坐下 | 精简 |
| 站了起来 | 站起 | 精简 |

## 四、对话词替换

| 原词 | 替换为 | 备注 |
|------|--------|------|
| 说道/开口说 | 说/道 | 能省就省 |
| 回答道/回应道 | 答/回 | 精简 |
| 询问道/问道 | 问 | 精简 |
| 心里想到 | 改内心独白或删除 | 避免冗余 |
| 他觉得 | 删除，直接写想法 | 更直接 |

---

## 五、状态词替换（Show Don't Tell）

| 原词 | 替换为 | 备注 |
|------|--------|------|
| 感到很悲伤 | 删除，用行动展示 | 如：他低下头，眼眶发红 |
| 气氛很紧张 | 删除，用细节展示 | 如：没人说话。有人后退一步。 |
| 心里很愤怒 | 删除，用行动展示 | 如：他把杯子摔在地上。 |
| 感到很害怕 | 删除，用动作展示 | 如：他的手在发抖。 |

## 六、AI 套话替换（必须删除）

| 原词 | 处理 | 原因 |
|------|------|------|
| 那一刻 | 删除或替换 | AI 套话 |
| 不可思议的是 | 删除 | AI 套话 |
| 宛如一幅画卷 | 删除 | AI 套话 |
| 交织在一起 | 删除 | AI 套话 |
| 见证了 | 删除或替换 | AI 套话 |
| 深吸了一口气 | 删除或换动作 | AI 套路 |
| 眼中闪烁着光芒 | 用具体眼神描述 | 脸谱化 |
| 嘴角勾起一抹微笑 | 删除或换表达 | 脸谱化 |
| 命运的齿轮 | 删除 | AI 烂文标志 |
| 仿佛……一般 | 限制使用 | 最多 1 个/段 |

## 七、脸谱化表情替换
