# 防卡文破局库

> 20种卡文解决方案

---

## 一、剧情卡文（10种）

### 1. 时间跳跃
```
适用：不知道接下来写什么日常/修炼
操作：直接跳到下一个关键节点
示例：
- "三年后。"
- "一个月后，中忍考试开始。"
```

### 2. 引入新角色
```
适用：当前角色关系无法推进
操作：新配角/新敌人/新势力登场
示例：
- 新的竞争者出现
- 新的任务目标
- 新的敌人出现
```

### 3. 意外事件
```
适用：剧情太平淡
操作：突发任务/突发事件打破日常
示例：
- 突然收到任务
- 村子被袭击
- 有人失踪
```

### 4. 信息爆炸
```
适用：需要推进主线但找不到切入点
操作：揭露一个重要秘密/真相
示例：
- 发现敌人的真实身份
- 获得关键情报
- 揭露历史真相
```

### 5. 强敌出现
```
适用：主角太强，缺乏对手
操作：引入实力碾压的新敌人
示例：
- 更强的敌人登场
- 新的威胁出现
- 原本的敌人变强
```

### 6. 资源获取
```
适用：主角需要成长但不知道怎么写
操作：新金手指/新能力/新装备
示例：
- 发现新功法
- 获得新武器
- 解锁新能力
```

### 7. 势力冲突
```
适用：需要扩张剧情
操作：与其他势力发生矛盾
示例：
- 势力被挑衅
- 抢夺地盘
- 资源争夺
```

### 8. 时间压力
```
适用：剧情拖沓
操作：给主角设置deadline
示例：
- "三天内必须完成"
- "比赛明天开始"
- "敌人将在24小时后到达"
```

### 9. 角色关系变化
```
适用：角色关系停滞
操作：结盟/背叛/收徒
示例：
- 敌人变盟友
- 盟友背叛
- 新的徒弟加入
```

### 10. 突发危机
```
适用：需要增加紧张感
操作：主角或重要角色陷入危险
示例：
- 被暗杀
- 中毒/受伤
- 被困
```

## 二、细节卡文（5种）

### 11. 跳过场景
```
适用：某个场景写不下去
操作：直接跳到下一个场景
示例：
- 战斗写不完 → 直接写结果
- 对话写不完 → 直接跳到结论
```

### 12. 换视角
```
适用：主角视角卡住
操作：切换到其他角色视角
示例：
- 主角视角 → 配角视角
- 我方视角 → 敌方视角
```

### 13. 时间倒叙
```
适用：需要补充背景
操作：用回忆/闪回方式补充信息
示例：
- "三天前……"
- "他想起了那一天……"
```

### 14. 并行线
```
适用：主线卡住
操作：切换到副线
示例：
- 主角卡住 → 写配角的行动
- 主线卡住 → 写敌人的计划
```

### 15. 减法写作
```
适用：场景太复杂
操作：删减不必要的元素，聚焦核心
示例：
- 多人场景 → 两人场景
- 复杂动作 → 简单一招
```

## 三、心理卡文（5种）

### 16. 回查大纲
```
操作：
1. 打开outline.md
2. 确认当前进度
3. 找到下一个事件
4. 直接写那个事件
```

### 17. 回查人设
```
操作：
1. 打开characters.md
2. 确认角色性格/行事原则
3. 问：这个角色会怎么做？
4. 写他/她会做的事
```

### 18. 用钩子库
```
操作：
1. 打开resource/剧情钩子库.md
2. 选一个开篇钩子
3. 直接用那个钩子开篇
4. 让钩子带着你写
```

### 19. 设置小目标
```
操作：
1. 给这一章设置一个小目标
2. 如：打一场、收一个人、解决一个问题
3. 只写这个目标
```

### 20. 先写结尾
```
操作：
1. 先确定这一章的结尾
2. 从结尾往回推
3. 写到结尾就停
```

## 四、卡文诊断清单

```
卡文位置：[第X章]
卡文原因：
□ 没有明确的下一个目标
□ 当前场景拖太长
□ 爽点类型单一
□ 角色关系没有推进
□ 没有外部压力
□ 不知道角色会怎么做

选择破局方案：
□ 时间跳跃到[下一个节点]
□ 引入[新角色/新事件]
□ 设置[新危机/新目标]
□ 回查大纲/人设
□ 用钩子库开篇
```
