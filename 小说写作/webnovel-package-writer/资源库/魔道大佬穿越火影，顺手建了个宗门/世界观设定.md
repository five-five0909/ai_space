## 一、世界观核心规则

### 体系融合设定

**查克拉本质**：查克拉 = 修仙界最原始的灵气，低级但兼容性高

**两大体系对比：**

| 对比项 | 忍术体系 | 修仙体系 |
|--------|----------|----------|
| 能量来源 | 查克拉 (身体能量 + 精神能量) | 天地灵气 + 自身修为 |
| 修炼方式 | 结印引导查克拉流动 | 经脉运转 + 功法配合 |
| 瓶颈 | 血统决定上限 | 功法决定上限 |
| 战斗方式 | 忍术对轰 | 意念操控 + 招式碾压 |
| 副作用 | 过度使用查克拉会死 | 无副作用，越战越强 |

**仙术的致命缺陷与破解：**
- 原著缺陷：需静止吸能、易石化、门槛高
- 主角方案：炼体抗石化 + 御灵边打边吸
- 效果：仙人模式？不，这是仙体常态

## 二、穿越时间点和触发原因

### 穿越原因
寿元将尽冲击最高境界，肉身崩解时撞上波风水门的空间波动，残魂穿界。

九尾之乱后，鸣人 12 岁（故事开始时）

## 七、混沌道尊专属表现规则

### 碾压规则
```
高一阶：一招秒杀，对方无法反应
高半阶：三招内结束战斗
同阶：对方用全力，主角随手
```

### 外化反应四级体系
| 级别 | 适用场景 | 表现 |
|------|----------|------|
| 一级外化 | 日常碾压 | 对方话说一半停住、下意识后退半步 |
| 二级外化 | 中等碾压 | 瞳孔骤缩、身体僵住、动物本能逃离 |
| 三级外化 | 重大碾压 | 自然环境异变、多个强者同时感知 |
| 四级外化 | 终极碾压 | 大范围自然规则紊乱、所有感知系能力者窒息 |

## 九、陌道宗设定

### 宗门核心
| 项目 | 内容 |
|------|------|
| 名称 | 陌道宗 |
| 宗主 | 陌尘（陌道尊） |
| 地点 | 木叶村外→独立驻地→忍界第三势力 |
| 信物 | 腰牌刻字"陌道无界" |

### 宗门规矩（唯一一条）
> "进了门，生死我担。"
> ——背叛者，死。

### 收徒标准
| 标准 | 要求 |
|------|------|
| 忠诚 | 绝对忠诚，不得背叛 |
| 天赋 | 有特殊才能或血继限界 |
| 因果 | 与主角有因果牵连 |

### 宗门发展线
| 阶段 | 章节 | 规模 | 地位 |
|------|------|------|------|
| 初建 | 第 50 章 | 3-5 人 | 木叶外围势力 |
| 发展 | 第 100 章 | 10+ 人 | 忍界知名势力 |
| 壮大 | 第 150 章 | 30+ 人 | 凌驾五影 |
| 巅峰 | 第 200 章 | 百人 + | 忍界霸主 |
